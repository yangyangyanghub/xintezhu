# 差异系数

> 差异系数（Coefficient of Variation, CV），又称**变异系数**，是衡量一组数据离散程度的相对指标。在教育均衡评估中，用于衡量区域内各学校某项指标的差异水平。

---

## 定义

$$CV = \frac{\sigma}{\bar{X}}$$

其中：
- $\sigma$ = 标准差 = $\sqrt{\frac{\sum(X_i - \bar{X})^2}{n-1}}$
- $\bar{X}$ = 平均值 = $\frac{\sum X_i}{n}$
- $n$ = 样本数量（区域内学校数）
- $X_i$ = 第 $i$ 所学校某项指标值

---

## 特点

| 特点 | 说明 |
|------|------|
| **无量纲** | 消除了量纲影响，可以在不同指标间直接比较 |
| **相对指标** | 用标准差与均值的比值来衡量离散程度 |
| **值越小越均衡** | CV 值越小，说明区域内各校该项指标越接近 |

---

## 教育领域标准阈值

| 场景 | 阈值 | 依据 |
|------|:----:|------|
| 优质均衡 - 小学 | CV ≤ 0.50 | 教督〔2017〕6号 |
| 优质均衡 - 初中 | CV ≤ 0.45 | 教督〔2017〕6号 |
| 质量监测综合系数 | < 0.15 | 教基〔2021〕3号 |
| 基本均衡 - 小学（旧） | CV ≤ 0.65 | 已废止 |
| 基本均衡 - 初中（旧） | CV ≤ 0.55 | 已废止 |

> **演进**：从基本均衡→优质均衡，差异系数标准收紧了约 18-23%。

---

## 与相关概念的关系

| 概念 | 关系 | 区别 |
|------|------|------|
| [[教育资源评估体系]] | 差异系数是优质均衡评估的核心计算工具 | 评估体系是制度框架，差异系数是具体计算方法 |
| 教育基尼系数 | 同属公平性测量工具 | 基尼系数侧重整体分布不均程度，差异系数侧重均值偏差的相对程度 |
| 泰尔指数 | 同属公平性测量工具 | 泰尔指数可分解为组内+组间差距，差异系数无此能力 |
