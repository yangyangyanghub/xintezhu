# 教育系统数据资源

> 小学教育数据统计表结构分析：247列7大模块318指标，10张数据库表设计建议。

---

## 数据规模

| 维度 | 数量 |
|------|------|
| 总列数 | 247 列 |
| 总指标 | 318 个 |
| 模块数 | 7 大模块 |

---

## 7大模块结构

| 模块 | 核心内容 |
|------|---------|
| 学校基础信息 | 学校编码、名称、类型、地址、办学层次 |
| 学生数据 | 在校生数、入学率、巩固率、随迁子女、留守儿童 |
| 师资数据 | 专任教师数、学历结构、骨干教师、交流轮岗 |
| 校舍与设备 | 教学用房面积、运动场馆、仪器设备值、多媒体教室 |
| 经费数据 | 生均公用经费、培训经费、特殊教育经费 |
| 评估指标 | 31项优质均衡指标 + 6项一票否决 |
| 质量监测 | 9学科监测结果、综合系数、差异系数 |

---

## 数据库表设计建议（10张表）

| 表名 | 用途 | 关联依据 |
|------|------|---------|
| `school_basic` | 学校基础信息 | 教督〔2017〕6号 |
| `student_data` | 学生统计 | 入学率/巩固率/随迁子女 |
| `teacher_data` | 师资统计 | 学历/骨干/音体美/交流轮岗 |
| `facility_data` | 校舍设备 | 面积/仪器设备/多媒体 |
| `finance_data` | 经费统计 | 生均经费/培训经费 |
| `evaluation_resource_config` | 资源配置7项评估 | 教督〔2017〕6号 |
| `evaluation_government_guarantee` | 政府保障15项评估 | 教督〔2017〕6号 |
| `evaluation_education_quality` | 教育质量9项评估 | 教基〔2021〕3号 |
| `evaluation_veto_record` | 一票否决记录 | 教督〔2017〕6号 |
| `quality_monitoring` | 质量监测结果 | 教督〔2021〕2号 |

---

## 关键计算字段

| 字段 | 计算公式 | 应用 |
|------|---------|------|
| `cv_ratio` | σ / X_bar | 7项资源配置差异系数 |
| `gini_coefficient` | 1 - Σ Qi × (2Ii - Wi) | 资源配置公平性 |
| `theil_index` | (1/n) × Σ(Yik/Y_bar) × ln(Yik/Y_bar) | 组内/组间差距分解 |
| `consolidation_rate` | (毕业生-三年转入+三年转出)/(三年前初一在校生-三年死亡) × 100% | 初中三年巩固率 ≥95% |
| `quality_comprehensive_coeff` | 良好及以上学生占比 | ≥0.5（2026年新规） |
