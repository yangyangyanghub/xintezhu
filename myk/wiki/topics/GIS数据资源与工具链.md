## 典型工作流

### 土地利用分析

```
数据下载 → 辐射定标 → 大气校正 → 影像分类 → 精度验证 → 制图出图
```

### 坐标转换

```
原始坐标 → 确定原坐标系 → 选择转换参数（四参/七参） → CGCS2000 → 精度检查
```

### 容积率统计

```
建筑面 → 建筑面积求和 → 用地面积 → 容积率 = 总建筑面积/用地面积
```
