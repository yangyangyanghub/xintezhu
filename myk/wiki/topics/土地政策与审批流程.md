# 土地政策与审批流程

> 从立项到竣工——工程建设项目全生命周期的土地审批与合规管理。

---

## 项目审批全流程

```
立项 → 选址 → 用地预审 → 报批 → 环评 → 用地规划许可证 → 施工 → 土地征收 → 供地 → 土地登记
```

### 关键环节

| 环节 | 说明 |
|------|------|
| **立项** | 政府投资项目/企业投资项目区分 |
| **规划选址** | 符合国土空间规划 |
| **用地预审** | 建设项目用地办理 |
| **用地报批** | 建设用地审批 |
| **环评** | 环境影响评价 |
| **土地征收** | 征收程序与补偿 |

### 全流程要点

- **速收藏**：2025 政府投资项目审批全流程图解，8 大环节一文理清
- **工程建设项目全流程图**：立项、选址、用地预审、报批、环评、用地规划许可证、土地征收、供地、土地登记
- **河北省建设项目规划许可管理办法**（试行）
