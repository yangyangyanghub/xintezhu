# Python语法学习笔记

## 1. Python基本语法元素

### 程序的格式框架

- **缩进**：Python 语言的语法元素之间通过缩进包含关系分层
  - 缩进是语法的一部分，缩进不正确会导致运行错误
  - 单层缩进：一个Tab或4个空格
- **注释**：以 `#` 开头，行为注释
- **多行注释**：以三个引号开头和结尾

### 变量与保留字

- **变量**：程序中用于保存和表示数据的占位符
  - 变量采用标识符（名字）来表示
  - 使用等号（=）进行赋值和修改值
- **命名规则**：
  - 大小写字母、数字、下划线和中文等字符
  - 大小写敏感，首字符不能是数字，不与保留字相同
- **保留字**：35个关键字，在程序内部定义并使用的标识符

### 数据类型

- **字符串**：由0个或多个字符组成的有序字符序列
  - 一对单引号或双引号表示
  - 字符串序号从0开始，反向递减从-1开始
  - 使用 `[]` 获取字符串中的一个或多个字符
  - 切片：`<字符串>[M:N]`

### 语句与函数

- **赋值语句**：由赋值号构成的一行代码
- **分支语句**：由判断条件决定程序运行方向的语句
- **函数**：根据输入参数产生不同输出的功能过程
  - 使用方式：`<函数名>(<参数>)`

## 2. 数字类型

### 整数类型

- 与数学中的整数概念一致
- 可正可负，没有取值范围限制
- 进制表示：
  - 二进制：以 `0b` 或 `0B` 开头
  - 八进制：以 `0o` 或 `0O` 开头
  - 十六进制：以 `0x` 或 `0X` 开头

### 浮点数类型

- 与数学中实数的概念一致
- 带小数点及小数的数字
- 取值范围和小数精度都存在限制
- 使用科学计数法表示大数

### 复数类型

- 实部和虚部都是浮点数
- 可用 `z.real` 获取实部，`z.imag` 获取虚部

### 数值运算符

| 运算符 | 使用 | 描述 |
|--------|------|------|
| + | x + y | 和 |
| - | x - y | 差 |
| * | x * y | 积 |
| / | x / y | 商 |
| // | x // y | 整数商（10//3 结果是3） |
| % | x % y | 余数（模运算） |
| ** | x ** y | 幂运算 |

### 数值运算函数

| 函数 | 描述 |
|------|------|
| `abs(x)` | 绝对值 |
| `divmod(x, y)` | 商余 |
| `pow(x, y)` | 幂运算 |
| `round(x, d)` | 四舍五入 |
| `max(x1, x2, ...)` | 最大值 |
| `min(x1, x2, ...)` | 最小值 |

## 4. time库使用

### 概述

time库是 Python 中处理时间的标准库：
- 计算机时间的表达
- 提供获取系统时间并格式化输出
- 提供系统级精确计时功能

### 时间获取函数

| 函数 | 描述 |
|------|------|
| `time()` | 获取当前时间戳 |
| `ctime()` | 获取当前时间字符串 |
| `gmtime()` | 获取当前时间结构体 |

### 时间格式化

| 格式化字符串 | 含义 |
|-------------|------|
| `%Y` | 年份 |
| `%m` | 月份 |
| `%d` | 日期 |
| `%H` | 小时（24小时制） |
| `%M` | 分钟 |
| `%S` | 秒 |

```python
import time
t = time.gmtime()
time.strftime("%Y-%m-%d %H:%M:%S", t)
```

### 程序计时应用

- **计时函数**：`perf_counter()` 获取精确计时
- **休眠函数**：`sleep()` 程序休眠指定秒数

```python
import time
start = time.perf_counter()
# 执行代码
end = time.perf_counter()
print(f"耗时: {end - start}秒")
```

## 5. 程序控制结构

### 单分支结构

```python
if <条件>:
    <语句块>
```

### 二分支结构

```python
if <条件>:
    <语句块1>
else:
    <语句块2>
```

### 多分支结构

```python
if <条件1>:
    <语句块1>
elif <条件2>:
    <语句块2>
...
else:
    <语句块N>
```

### 条件判断及组合

| 操作符 | 描述 |
|--------|------|
| `and` | 逻辑与 |
| `or` | 逻辑或 |
| `not` | 逻辑非 |

### 异常处理

```python
try:
    <语句块1>
except <异常类型>:
    <语句块2>
else:
    <语句块3>  # 不发生异常时执行
finally:
    <语句块4>  # 一定执行
```

## 6. 循环结构

### 遍历循环

```python
for <循环变量> in <遍历结构>:
    <语句块>
```

**计数循环**：
```python
for i in range(N):   # 0 到 N-1
    <语句块>

for i in range(M, N):  # M 到 N-1
    <语句块>
```

### 无限循环

```python
while <条件>:
    <语句块>
```

### 循环控制保留字

- `break`：跳出当前循环
- `continue`：跳过本次循环

### 循环的高级用法

- `for-else`：循环正常结束时执行 else 块
- `while-else`：循环正常结束时执行 else 块

```python
for i in range(5):
    if i == 3:
        break
else:
    print("循环正常结束")  # break 时不执行
```

## 7. random库

random库是使用随机数的 Python 标准库。

### 基本随机数函数

| 函数 | 描述 |
|------|------|
| `seed(a)` | 设置随机种子 |
| `random()` | 生成 [0.0, 1.0) 随机小数 |

### 扩展随机数函数

| 函数 | 描述 |
|------|------|
| `randint(a, b)` | 生成 [a, b] 随机整数 |
| `randrange(a, b, k)` | 生成 [a, b) 步长为 k 的随机整数 |
| `getrandbits(k)` | 生成 k 比特长随机整数 |
| `uniform(a, b)` | 生成 [a, b] 随机小数 |
| `choice(seq)` | 从序列随机选择一个元素 |
| `shuffle(seq)` | 随机打乱序列 |
| `sample(seq, k)` | 随机选择 k 个元素 |

### 使用示例

```python
import random

# 设置随机种子，保证结果可复现
random.seed(10)

# 生成随机整数
num = random.randint(1, 100)

# 从列表随机选择
items = ['a', 'b', 'c', 'd']
choice = random.choice(items)

# 随机打乱列表
random.shuffle(items)
```

## 8. 函数定义与使用

### 函数定义

```python
def <函数名>(<参数列表>):
    <函数体>
    return <返回值>
```

### 参数传递

- **位置参数**：按位置顺序传递
- **关键字参数**：按参数名传递
- **默认参数**：定义时设置默认值
- **可变参数**：`*args` 接收任意数量位置参数，`**kwargs` 接收任意数量关键字参数

```python
def func(a, b, c=10, *args, **kwargs):
    pass

func(1, 2)                # a=1, b=2, c=10
func(1, 2, 3)             # a=1, b=2, c=3
func(1, 2, c=20)          # a=1, b=2, c=20
func(1, 2, 3, 4, 5)       # a=1, b=2, c=3, args=(4, 5)
func(1, 2, x=10, y=20)    # a=1, b=2, c=10, kwargs={'x':10, 'y':20}
```

### lambda 函数

```python
# lambda 匿名函数
f = lambda x, y: x + y
print(f(1, 2))  # 输出 3

# 常见用法
sorted(lst, key=lambda x: x[1])  # 按元组第二元素排序
```

## 10. 代码规范建议

### PEP 8 规范要点

- 使用 4 个空格缩进
- 每行最大长度 79 字符
- 类名使用驼峰命名法（CamelCase）
- 函数名使用下划线命名法（snake_case）
- 常量使用全大写
- 运算符两侧各加一个空格
- 逗号后加一个空格

### 注释规范

```python
# 单行注释

"""
多行注释
用于文档字符串
"""

def func():
    """函数文档字符串"""
    pass
```
