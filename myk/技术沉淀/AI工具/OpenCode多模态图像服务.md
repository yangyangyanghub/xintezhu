## 一、核心能力矩阵

| 能力 | 说明 | 典型场景 |
|------|------|----------|
| 文生图 | 根据文字描述生成图片 | 文章配图、海报、信息图、封面 |
| 图生图 | 在已有图片基础上编辑 | 换背景、加元素、改风格、局部修改 |
| 图生文 | 图片描述、OCR识别、图表分析 | 截图识字、数据提取、图表解读 |
| 长图拼接 | 多张图片垂直拼接 | 微信长图、教程步骤、早报海报 |

## 二、Skill 协同场景

| 协同组合 | 场景 | 产出 |
|----------|------|------|

## 三、image-service 技能结构

```
image-service/
├── SKILL.md              # 技能定义
├── scripts/              # 可执行脚本
│   ├── text_to_image.py  # 文生图
│   ├── image_to_image.py # 图生图
│   ├── image_to_text.py  # 图生文
│   └── merge_long_image.py # 长图拼接
├── config/
│   └── settings.json     # API配置
└── references/           # 参考资料
    ├── long-image-guide.md
    └── prompt-guide.md
```

### API 配置示例

```json
{
  "IMAGE_API_BASE_URL": "https://api.openai.com/v1",
  "IMAGE_API_KEY": "你的API Key",
  "IMAGE_MODEL": "dall-e-3",
  "VISION_MODEL": "gpt-4o"
}
```

支持的 API：OpenAI（DALL-E 3、GPT-4o）、通义万相、智谱 CogView、文心一格

## 四、提示词技巧

### 基本结构

```
[主体内容] + [风格] + [色调] + [细节要求]
```

### 好的示例

```
一张16:9横版技术教程封面图，数字萌系风格，像素块元素+马卡龙渐变色调，
模块圆润Q弹像软胶玩具，边缘柔软没有锋利线条。
背景为低饱和马卡龙渐变（薄荷绿到淡紫色），画面中央是一个可爱的机器人助手形象，
顶部大标题：玩转OpenCode。文字全部为中文
```

### 风格关键词

| 风格 | 关键词 |
|------|--------|
| 科技感 | 科技感、未来风、赛博朋克、霓虹、深色背景 |
| 可爱风 | 萌系、Q弹、软胶、马卡龙、圆润 |
| 商务风 | 简约、专业、大气、高级感、蓝灰色调 |
| 手绘风 | 手绘、插画、线条、水彩、素描 |

### 宽高比

| 比例 | 适用场景 |
|------|----------|
| 1:1 | 头像、图标 |
| 3:4 | 小红书、竖版海报 |
| 4:3 | 传统显示器、PPT |
| 16:9 | 横版Banner、封面 |
| 9:16 | 手机壁纸、短视频封面 |

## 五、实战案例速查

### 案例1：批量配图

```
阅读这篇文章，为每个章节生成一张16:9的配图：
- 风格：数字萌系，马卡龙色调
- 要求：每张图体现该章节的核心内容
- 输出到：images/ 目录
```

### 案例2：AI新闻早报

```
抓取今天的AI新闻，生成一张竖版早报长图：
- 顶部：日期标题
- 中间：5条核心新闻，每条配小图标
- 底部：来源说明
- 风格：科技感，深色背景
```

### 案例3：儿童绘本

```
根据这个故事生成绘本插图：
故事：小兔子找萝卜
第1页：小兔子在森林里醒来
第2页：小兔子遇到了小松鼠
...
要求：
- 风格：可爱水彩插画，柔和色调
- 角色一致：小兔子是白色的，穿蓝色背带裤
```

## 六、命令参考

### 文生图

```bash
python .opencode/skills/image-service/scripts/text_to_image.py \
  "信息图风格，标题：AI技术趋势，科技感蓝紫配色" \
  -r 16:9 \
  -o output.png
```

### 图生文

```bash
python .opencode/skills/image-service/scripts/image_to_text.py \
  screenshot.png \
  -m ocr  # describe | ocr | chart | scene
```

### 长图拼接

```bash
python .opencode/skills/image-service/scripts/merge_long_image.py \
  img1.png img2.png img3.png \
  -o long.png \
  --blend 20
```

## 七、常见问题

| 问题 | 解决方案 |
|------|----------|
| 生成失败 | 检查API配置、余额、敏感词 |
| OCR不准 | 提高图片清晰度、裁剪文字区域、指定语言 |
| 风格不一致 | 角色描述要具体、先生成设定图确认、每次都带完整描述 |
| 中文不好看 | 提示词加"中文字体清晰"、减少图片内文字 |

---

## 八、关键要点

1. **提示词要具体**：风格、色调、细节写清楚
2. **批量生成保持统一**：固定风格描述，每次都写完整
3. **长图需求识别**：提到"长图"、多段落、3+独立画面
4. **Skill 协同威力大**：一句话串联多个能力，输出完整产品
