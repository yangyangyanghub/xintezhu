# 登录页面 DESIGN.md

## 1. Visual Theme

- **Design Intent**: 安全可信、简洁高效的品牌登录体验。传达"专业、可靠、现代"的产品气质，让用户快速完成身份验证并进入核心业务区域
- **Style Keywords**: `minimalism`、`trustworthy`、`clean`、`professional`、`secure`、`focused`
- **Reference Notes**: 参考 Notion 登录页（极简布局 + 信任信号）、Stripe Dashboard 登录（品牌色调克制、表单区域突出）。避免过度装饰、避免使用卡通化元素、避免高饱和度渐变背景干扰表单焦点

## 2. Color Palette

- **Primary Colors**: 
  - 主操作色：品牌核心色（用于登录按钮、成功反馈图标）
  - 文字色：深灰/近黑（主标题、表单标签）、中灰（辅助说明）
  - 背景色：纯白或极浅灰（主背景区）
  
- **Secondary Colors**: 
  - 错误/警示色：红色系（校验失败反馈、错误提示边框）
  - 成功色：绿色系（登录成功后的状态指示）
  - 链接色：品牌色或蓝色系（忘记密码入口、第三方登录文字）
  - 禁用色：浅灰+降低不透明度（提交中按钮、未填写时的登录按钮）

- **Usage Rules**: 
  - 表单区域使用纯净背景，不叠加纹理或渐变
  - 反馈色仅用于状态提示，不作为装饰
  - 第三方登录图标使用原始品牌色，保持识别度
  - 所有颜色需满足 WCAG AA 对比度要求（文字 vs 背景 ≥ 4.5:1）

## 3. Typography

- **Type System**: 
  - H1 页面标题："登录" 或 "欢迎回来"（居中或左对齐，字号 24-32px）
  - H2 区块标题：第三方登录分隔区（字号 18-20px，可选）
  - Body 表单标签："邮箱地址"、"密码"、"记住我"（字号 14-16px）
  - Caption 辅助文字：错误提示、忘记密码说明（字号 12-14px）

- **Font Guidance**: 
  - 字体风格：无衬线字体，现代感优先（Inter、SF Pro、PingFang SC、思源黑体）
  - 字重：标题 500-600，标签 400，辅助 300-400
  - 可读性：行高 ≥ 1.5，避免字距过紧

- **Text Rules**: 
  - 表单标签与输入框间距：4-8px
  - 错误提示与输入框间距：4px
  - 单行文本最大行长：不限制（登录页为窄布局）
  - 多语言适配：预留文字溢出空间（中文/英文切换时高度变化）

## 4. Components

### Core Components

| 组件 | 功能 | 视觉处理 |
|------|------|---------|
| **邮箱输入框** | 用户身份识别入口 | 标准文本输入框，占位文字"请输入邮箱地址"，邮箱图标可选前置 |
| **密码输入框** | 密码验证入口 | 标准输入框 + 密码隐藏/显示切换图标，占位文字"请输入密码" |
| **记住我复选框** | 持久化登录选项 | 小尺寸复选框 + 文字标签"记住我"，默认不勾选 |
| **登录按钮** | 触发认证流程 | 品牌主色填充，文字"登录"或"立即登录"，高度 40-48px |
| **忘记密码链接** | 密码找回入口 | 文字链接，位置：密码输入框下方或表单底部，颜色为链接色 |
| **第三方登录占位** | 社交账号快捷登录 | 图标按钮组（Google、GitHub、微信等），分隔线 + "或使用其他方式登录" |

### States Design

#### 空态（Empty State）
- **邮箱输入框**: 占位文字显示"请输入邮箱地址"，边框为默认灰色
- **密码输入框**: 占位文字显示"请输入密码"，边框为默认灰色
- **登录按钮**: 禁用状态（浅灰背景 + 降低不透明度），不可点击，提示"请先填写信息"

#### 输入中（Typing State）
- **邮箱输入框**: 边框高亮（品牌色或蓝色），光标可见，输入文字实时显示
- **密码输入框**: 边框高亮，密码以掩码形式显示（●●●●●），眼睛图标可切换明文
- **登录按钮**: 若邮箱格式有效 + 密码长度≥最小值，按钮变为可点击状态

#### 校验失败（Validation Error）
- **邮箱输入框**: 边框变为错误色（红色），输入框下方显示错误提示（字号 12-14px，红色文字）
  - 错误类型示例：
    1. 格式错误："请输入有效的邮箱地址"
    2. 必填空："邮箱地址不能为空"
    3. 未注册："该邮箱未注册，请先注册账号"
- **密码输入框**: 边框变红，下方显示错误提示
  - 错误类型示例：
    1. 必填空："密码不能为空"
    2. 长度不足："密码长度至少为 6 位"
    3. 格式错误："密码需包含字母和数字"（若有此规则）
- **登录按钮**: 禁用状态，不响应点击
- **视觉反馈**: 输入框抖动动画可选（轻微位移 2-4px，持续 200ms）

#### 提交中（Submitting State）
- **登录按钮**: 
  - 文字变为"登录中..."或显示加载图标（圆圈旋转动画）
  - 背景色降低不透明度（例如 opacity: 0.7）
  - 禁用点击，防止重复提交
- **输入框**: 禁用状态，不可编辑，边框保持默认色
- **页面整体**: 可选遮罩层 + 中央加载动画（全屏锁定）

#### 成功后（Success State）
- **即时反馈**: 
  - 登录按钮文字变为"登录成功"或显示成功图标（绿色对勾）
  - 背景色变为成功色（绿色），持续 800-1200ms
- **页面跳转**: 
  - 选项 A：自动跳转到主页/仪表盘（无过渡动画）
  - 选项 B：显示"欢迎回来，xxx" 提示 + 手动跳转按钮
- **记住我生效**: 若勾选，浏览器存储凭证，下次自动填充

### Behavior Notes

- **表单验证时机**: 
  - 选项 A：实时验证（输入时立即校验格式）
  - 选项 B：提交时验证（点击登录按钮后统一校验）
  - 选项 C：混合模式（输入时校验格式，提交时校验业务规则如未注册）
  
- **密码显示切换**: 
  - 默认隐藏，眼睛图标可切换明文/掩码
  - 切换时图标从"闭眼"变为"睁眼"，输入框内容从掩码变为明文
  
- **记住我行为**: 
  - 勾选后保存登录状态到浏览器存储（localStorage 或 cookie）
  - 下次访问自动填充邮箱，密码是否填充需安全评估
  
- **忘记密码跳转**: 
  - 点击后跳转到独立找回密码页面或在当前页弹出模态框
  - 模态框内容：邮箱输入 + 发送重置邮件按钮

### Spacing System

- **容器宽度**: 登录表单区域宽度 360-400px（居中）
- **输入框高度**: 40-48px（触控友好）
- **输入框间距**: 垂直间距 16-24px
- **按钮上方间距**: 24-32px（与输入框分离）
- **按钮下方间距**: 16px（与忘记密码链接分离）
- **第三方登录区间距**: 分隔线上方 32px，下方 24px
- **图标与文字间距**: 前置图标 + 输入框文字间距 8-12px

### Content Flow

- **信息优先级**: 
  1. 页面标题（品牌认知）
  2. 邮箱输入框（首要填写）
  3. 密码输入框（次要填写）
  4. 登录按钮（核心操作）
  5. 忘记密码（辅助操作）
  6. 第三方登录（备选方案）
  7. 注册引导（转化入口）
  
- **阅读顺序**: 从上至下，视线自然流线（标题 → 输入框 → 按钮 → 链接 → 第三方）
- **关键路径**: 邮箱 → 密码 → 登录（主流程），忘记密码/第三方登录为分支路径

## 6. Depth

- **Elevation Strategy**: 
  - 登录表单区可选轻微阴影（0-2px elevation，营造卡片感）
  - 或与背景融为一体（扁平设计，无阴影）
  - 弹出模态框（忘记密码）使用高阴影层级（8-16px）
  
- **Borders and Surfaces**: 
  - 输入框边框：默认灰色（1px），聚焦时品牌色（2px），错误时红色（2px）
  - 登录按钮：无边框或品牌色边框（1px），填充背景
  - 第三方登录图标：无边框，保持原始图标样式
  
- **Motion Cues**: 
  - 输入框聚焦：边框颜色过渡（200ms ease）
  - 校验失败抖动：translateX 微移（200ms）
  - 登录按钮加载：旋转图标动画（持续循环）
  - 成功反馈：背景色渐变过渡（800ms）

## 7. Icon System

- **Icon Source**: 
  - 候选库：Lucide（轻量、一致性好）、Heroicons（兼容 Tailwind）、Material Icons（成熟）
  - 选择原则：线条图标优先，与表单线条风格一致；第三方品牌图标使用官方 SVG
  
- **Usage Guidelines**: 
  - 预置图标尺寸：16-20px（输入框内）或 24px（独立按钮）
  - 线条粗细：1.5-2px（与输入框边框一致）
  - 配色：默认灰色，聚焦时品牌色，错误时红色
  - 搭配规则：图标居左，文字居右，间距 8-12px
  
- **Conflict Handling**: 
  - 若使用多个图标库：品牌预置图标统一用 Lucide，第三方登录图标各用原生 SVG
  - 命名空间：预置图标命名 `icon-{name}`，第三方图标命名 `icon-{brand}-{name}`（如 icon-google-logo）

## 8. Responsive

- **Breakpoints**: 
  - 桌面：≥ 768px（表单区居中，宽度固定 360-400px）
  - 平板：768-1024px（表单区宽度略微缩小至 320-360px）
  - 手机：< 768px（表单区全宽，占满屏幕宽度的 85-90%）
  
- **Adaptation Rules**: 
  - 桌面/平板：品牌 Logo 居中或左对齐，表单区居中
  - 手机：品牌 Logo 居中（可选隐藏），表单区居中全宽
  - 第三方登录图标：桌面横向排列，手机可选纵向排列或保持横向但间距缩小
  
- **Priority Changes**: 
  - 桌面：所有元素完整显示
  - 手机：可选隐藏品牌 Logo 或注册引导文字，减少视觉干扰
  - 手机：第三方登录图标间距缩小，但保持点击区域 ≥ 44px
