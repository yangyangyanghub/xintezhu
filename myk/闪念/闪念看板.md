# 💡 闪念看板

> 自动汇总所有闪念卡片，按状态分组

---

## 📥 待处理闪念

```dataview
TABLE WITHOUT ID
  file.link AS "闪念",
  created AS "记录时间",
  tags AS "标签"
FROM "myk/闪念/inbox"
WHERE type = "flash" AND status = "inbox"
SORT created DESC
```

---

## 📦 已沉淀闪念

```dataview
TABLE WITHOUT ID
  file.link AS "闪念",
  created AS "记录时间",
  tags AS "标签"
FROM "myk/闪念/processed"
WHERE type = "flash" AND status = "processed"
SORT created DESC
```

---

## 🏷️ 按标签浏览

### 查看所有闪念标签

```dataview
TABLE WITHOUT ID
  key AS "标签",
  length(rows) AS "数量"
FROM "myk/闪念"
WHERE type = "flash"
FLATTEN tags
GROUP BY tags AS key
SORT length(rows) DESC
```

---

## 📊 统计

```dataview
TABLE WITHOUT ID
  "总计" AS "指标",
  length(filter(file.lists, (f) => true)) AS "数量"
FROM "myk/闪念"
WHERE type = "flash"
GROUP BY true
```
