# 极简海报设计

## 提示词

```
Design a minimalist poster for [主题]. 

Key elements:
- Clean typography with bold headline
- Limited color palette (max 3 colors)
- Generous white space
- Single focal point image
- Modern sans-serif fonts

Style: Minimalist, professional, elegant
Mood: Sophisticated yet approachable
```

## 参数建议

| 参数 | 建议值 |
|------|--------|
| 比例 | 3:4 或 9:16 |
| 风格 | Minimalist |
| 质量 | High |

## 适用场景

- 产品发布会海报
- 品牌宣传活动
- 艺术展览
- 高端服务推广

## 变体版本

### 深色版
在提示词中加入：`dark background, dramatic lighting`

### 亮色版
在提示词中加入：`bright background, fresh and clean`
