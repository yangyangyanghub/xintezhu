# 信息图表设计

## 提示词

```
Create an infographic about [主题/数据].

Design elements:
- Clear hierarchy and flow
- Icon-based visual representation
- Data visualization (charts, graphs, percentages)
- Number highlights with large typography
- Color-coded sections
- Step-by-step progression or comparison layout

Style: Modern flat design, professional, data-driven
Colors: Corporate palette with accent colors
Layout: Vertical scroll-friendly or horizontal timeline
```

## 参数建议

| 参数 | 建议值 |
|------|--------|
| 比例 | 9:16 (长图) 或 16:9 |
| 风格 | Flat design, infographic |
| 元素 | Icons, charts, numbers |

## 常见类型

### 时间线
`timeline infographic, chronological order, milestone markers`

### 对比图
`comparison infographic, side by side, pros and cons`

### 流程图
`process infographic, step by step, workflow visualization`

### 数据统计
`statistics infographic, percentage circles, bar charts, key metrics`

## 使用建议

1. 先整理好要展示的数据点
2. 确定信息层级（主标题、副标题、数据点）
3. 选择合适的图表类型
4. 保持色彩一致性
