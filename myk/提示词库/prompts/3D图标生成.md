# 3D 图标生成

## 提示词

```
Create a 3D icon of [对象], 

Style specifications:
- Soft pastel colors
- Smooth gradients
- Subtle shadows and highlights
- Glossy or matte finish
- Isometric or slightly tilted angle
- Clean background (white or light gradient)

Technical:
- High detail
- Smooth edges
- Professional lighting
- 3D rendered look
- No text
```

## 参数建议

| 参数 | 建议值 |
|------|--------|
| 比例 | 1:1 |
| 风格 | 3D Render |
| 背景 | Clean white |
