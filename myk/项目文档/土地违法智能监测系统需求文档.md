## 一、项目概述

### 1.1 项目背景

当前土地违法监测主要依赖人工巡查和卫星遥感比对，存在以下问题：

| 痛点 | 影响 |
|------|------|
| 巡查效率低 | 人工巡查覆盖面有限，难以及时发现违法 |
| 发现周期长 | 卫星影像更新周期 7-30 天，违法行为已成事实 |
| 证据收集难 | 现场取证需要大量人力，证据链不完整 |
| 处置响应慢 | 发现到处置流程长，违法主体易逃脱 |

### 1.2 项目目标

构建一套**实时、智能、自动化**的土地违法监测系统，实现：

- **实时监测**：无人机视频流实时分析，发现违法立即告警
- **智能识别**：AI 自动识别违法建设、挖掘取土、耕地占用等行为
- **证据留存**：自动生成违法证据链，满足执法要求
- **系统对接**：与国土管理系统打通，实现闭环处置

### 1.3 预期效果

| 指标 | 现状 | 目标 |
|------|------|------|
| 发现周期 | 7-30 天 | 实时 |
| 巡查效率 | 5 平方公里/人/天 | 50 平方公里/架次/天 |
| 违法发现率 | 约 60% | > 90% |
| 证据完整性 | 约 50% | > 95% |

## 二、用户角色

### 2.1 角色定义

| 角色 | 职责 | 使用场景 |
|------|------|---------|
| **巡查人员** | 驾驶无人机巡查、现场核实 | 日常巡查、应急响应 |
| **监测中心人员** | 接收告警、研判分析、派发任务 | 监测中心值班 |
| **执法人员** | 现场执法、取证固证 | 违法处置 |
| **管理人员** | 查看统计报表、决策支持 | 领导决策 |
| **系统管理员** | 系统配置、数据维护 | 后台管理 |

### 2.2 用户故事

**巡查人员**：
> 作为巡查人员，我希望无人机飞行时系统能自动识别违法，这样我可以专注于飞行安全。

**监测中心人员**：
> 作为监测中心人员，我希望违法线索能自动推送并附带证据，这样我可以快速研判和派发。

**执法人员**：
> 作为执法人员，我希望系统能提供完整的证据链，这样我可以依法处置。

**管理人员**：
> 作为管理人员，我希望看到辖区违法态势统计，这样我可以做出决策。

### 3.2 核心功能详细说明

#### 3.2.1 无人机视频接入

| 功能项 | 说明 |
|------|------|
| 视频流接入 | 支持 RTSP、WebRTC 等主流协议 |
| 多机接入 | 同时接入多架无人机视频流 |
| 视频分发 | 支持多终端同时观看 |
| 云台控制 | 支持远程控制无人机云台 |
| 飞行状态 | 实时显示无人机位置、高度、电量等 |

**输入**：无人机视频流 + 飞行数据  
**输出**：实时画面 + 飞行状态

#### 3.2.2 实时变化检测

| 功能项 | 说明 |
|------|------|
| 基准影像加载 | 加载历史正射影像作为基准 |
| 实时比对 | 实时视频帧与基准影像比对 |
| 变化区域标注 | 自动标注变化区域边界 |
| 变化面积计算 | 计算变化区域面积 |
| 变化类型判断 | 初步判断变化类型（建设/挖掘/堆放等） |

**输入**：实时视频帧 + 基准影像 + 相机参数  
**输出**：变化区域 + 面积 + 类型

**算法要求**：
- 变化检测准确率 > 85%
- 单帧处理时间 < 500ms
- 支持阴影、季节变化鲁棒

#### 3.2.3 违法目标识别

| 功能项 | 说明 |
|------|------|
| 建筑物检测 | 检测新建、在建、扩建建筑物 |
| 挖掘检测 | 检测取土坑、基坑等 |
| 堆放物检测 | 检测建筑垃圾、砂石堆等 |
| 设施检测 | 检测临时工棚、围挡等 |
| 违法判断 | 叠加土地利用规划，判断是否违法 |

**输入**：变化检测结果 + 土地利用数据  
**输出**：违法目标 + 类型 + 位置

**识别类型**：

| 违法类型 | 定义 | 风险等级 |
|---------|------|---------|
| 违法建设 | 在非建设用地上新建建筑物 | 高 |
| 违法挖掘 | 非法取土、采砂、挖塘 | 高 |
| 耕地占用 | 在耕地上进行非农建设 | 高 |
| 临时违建 | 未经审批搭建临时设施 | 中 |
| 堆放占用 | 临时堆放建筑材料等 | 低 |

#### 3.2.4 风险等级评估

| 功能项 | 说明 |
|------|------|
| 面积评估 | 根据违法面积评估风险 |
| 地类评估 | 根据占用地类评估风险 |
| 规划评估 | 根据规划用途评估风险 |
| 综合评分 | 多维度综合评分 |

**风险等级标准**：

| 等级 | 条件 | 响应时限 |
|------|------|---------|
| **高风险** | 基本农田任何占用 / 耕地占用 > 1 亩 / 重点区域违法 | 2 小时 |
| **中风险** | 耕地占用 0.5-1 亩 / 林地占用 / 一般区域违法 | 24 小时 |
| **低风险** | 其他用地变更 / 面积 < 0.5 亩 | 72 小时 |

#### 3.2.5 告警推送

| 功能项 | 说明 |
|------|------|
| 实时推送 | 发现违法后秒级推送 |
| 多渠道推送 | 支持系统消息、短信、微信、APP 推送 |
| 分级推送 | 不同风险等级推送给不同人员 |
| 告警去重 | 同一区域重复告警自动合并 |

**推送规则**：

| 风险等级 | 推送对象 | 推送方式 |
|---------|---------|---------|
| 高风险 | 监测中心 + 分管领导 | 系统消息 + 短信 + 电话 |
| 中风险 | 监测中心 + 执法队长 | 系统消息 + 短信 |
| 低风险 | 监测中心 | 系统消息 |

#### 3.2.6 告警研判

| 功能项 | 说明 |
|------|------|
| 告警详情 | 查看违法位置、面积、类型、图片等 |
| 历史对比 | 查看该位置历史影像对比 |
| 规划查询 | 查询该位置土地利用规划 |
| 研判标注 | 人工标注研判意见 |
| 任务派发 | 派发给执法人员现场核实 |

#### 3.2.7 证据链生成

| 功能项 | 说明 |
|------|------|
| 时间戳 | 记录发现时间、核实时间等 |
| 位置信息 | GPS 坐标 + 地址描述 |
| 影像证据 | 现场照片、视频、航拍图 |
| 对比证据 | 违法前后对比影像 |
| 文档生成 | 自动生成违法线索报告 |

**证据链要求**：
- 时间：精确到秒
- 位置：经纬度精确到小数点后 6 位
- 图片：原图 + 标注图，保留 EXIF 信息
- 视频：关键片段截取，时长 10-30 秒
- 报告：符合国土执法文书规范

#### 3.2.8 巡查计划管理

| 功能项 | 说明 |
|------|------|
| 计划制定 | 制定巡查区域、频次、人员 |
| 重点区域标注 | 标注重点监测区域 |
| 航线规划 | 自动生成巡查航线 |
| 计划执行 | 派发给巡查人员执行 |
| 计划调整 | 根据实际情况调整计划 |
