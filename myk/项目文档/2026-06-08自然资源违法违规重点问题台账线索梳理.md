## 二、省厅工作安排

### 1. 省级问题台账下发

省厅将结合以下数据和渠道，形成省级集中整治问题台账，并点对点下发各地：

- 年度国土变更调查。
- 用地用矿审批数据。
- 舆情监测。
- 有奖举报。
- 来信来访。
- 12345 热线。
- 遥感影像、低空智能监管、视频监控等技术发现线索。

对纳入省级台账的问题，各地需实行全链条管理：

```text
清查 → 建档 → 查处 → 整改 → 追责 → 销号
```

原则要求：纳入台账的问题，确保 **6 月 30 日前基本整改到位**。

### 2. 省级核查组进驻核查

省厅将组建 **12 个核查组**，采取“四不两直”方式，对纳入台账的重点问题线索开展省级直接核查。

核查重点包括：

- 问题线索是否属实。
- 地方整治进度是否真实。
- 整改质量是否达标。
- 是否存在重大典型问题。
- 是否存在整改缓慢、敷衍塞责、弄虚作假等情况。

对核查发现的重大典型问题，省厅将采取以下措施：

- 公开通报。
- 挂牌督办。
- 专题报告省政府。
- 提请省领导组织约谈。

### 3. 省级办案组直接查办

省厅将成立土地、矿产两个省级办案组，由省厅执法人员带队，并以执法办案人员人才库为补充，对重点线索开展核查和典型案件查办。

会议明确“四个一律”：

1. 对查实的违法违规重点问题，一律从严从重查处到位。
2. 涉及地方党委政府和相关部门违规决策、监管不力、失职渎职、弄虚作假的，一律移交纪检监察机关追责问责。
3. 涉嫌犯罪的，一律移交公安机关追究刑事责任。
4. 对非法采矿案件，特别是以工程项目名义非法盗采的，一律提级查办，没收违法所得和非法采出矿产品，查封扣押工具设备。

### 4. 整治成果运用

集中整治期间，省厅将筛选以下问题纳入全省警示教育和通报范围：

- 重大典型问题。
- 顶风违法案件。
- 整改不力案件。
- 问题突出的地区。
- 线索高发、整治缓慢的重点地区。

成果运用方式包括：

- 全省通报。
- 拍摄警示片。
- 纳入全省警示教育会议。
- 要求问题突出地区作表态发言。
- 对整治缓慢地区进行重点调度和约谈。

### 5. 向党委政府专题汇报要求

会议要求各地会后第一时间向本级党委政府主要领导专题汇报，重点汇报：

- 本次集中整治目的。
- 省厅整治要求。
- 本地问题线索情况。
- 本地整改计划和责任分工。
- 从严整治、从严问责的工作信号。

### 6. 省市县三级责任分工

| 层级 | 工作责任 |
|---|---|
| 省厅 | 统筹抓总、定期调度、组织核查、直接办案、通报约谈、推动全省整治平衡开展 |
| 市局 | 承上启下，落实省厅要求，督促指导县级工作，对重点问题和重点地区加强盯办 |
| 县区 | 扛起一线责任，全面排查、逐项核实、逐案处置、逐个销号，防止新增问题发生 |

会议特别强调，要把整治工作的真实性、有效性摆在首位，坚决杜绝有案不立、压案不查，坚决杜绝敷衍塞责、弄虚作假。

## 三、统一问题台账模板

|---|---|

| `问题名称` | 简明描述 |
| `线索来源` | 遥感、低空监管、信访、舆情、12345、有奖举报、上级交办等 |
| `所在市` | 市级行政区 |
| `所在县区` | 县区 |
| `所在乡镇街道` | 乡镇街道 |
| `所在村/社区` | 村、社区 |
| `具体位置` | 地址、地块、矿点、小区等 |
| `坐标` | 经度、纬度 |
| `涉及面积` | 亩/平方米 |
| `涉及地类` | 耕地、永久基本农田、林地、建设用地、生态红线等 |
| `违法主体` | 企业、个人、村集体、政府项目、未知 |
| `问题描述` | 事实描述 |
| `初判违法类型` | 根据会议重点问题分类 |
| `发现时间` | 发现线索时间 |

| `是否新增问题` | 新增/存量/待核实 |
| `是否重点问题` | 是/否 |
| `是否典型案件` | 是/否 |

| `责任单位` | 县区政府、乡镇、部门 |
| `核查时间` | 核查日期 |
| `核查结论` | 属实/不属实/部分属实/待进一步核查 |
| `处置措施` | 立案、拆除、复耕、恢复原状、处罚、移交等 |
| `整改时限` | 原则上 6 月 30 日前 |
| `整改进度` | 未核查/核查中/整改中/已完成/已销号 |
| `是否行政立案` | 是/否 |
| `是否刑事移交` | 是/否 |
| `是否纪检监察移交` | 是/否 |
| `是否挂牌督办` | 是/否 |
| `销号材料` | 照片、处罚决定、验收意见等 |
| `备注` | 其他情况 |

## 五、整改销号流程

```text
线索接收 → 初步筛查 → 现场核查 → 问题认定 → 分类处置 → 整改实施 → 复核验收 → 销号归档 → 回头看
```

## 六、风险分级口径

- **高风险**：涉及永久基本农田、生态红线、非法采矿、违建别墅、违法住宅小区、群众入住或销售、媒体曝光、上级重点关注、安全消防隐患、弄虚作假等。
- **中风险**：涉及一般耕地、林地、设施农用地，违法面积较大但未形成重大舆情，处置路径相对清晰，需多部门协同。
- **低风险**：面积较小，不涉及红线底线，违法事实清楚，整改难度低，无明显舆情和信访风险。

## 七、后续使用建议

1. 先用统一主台账接收全部线索。
2. 再按问题类别建立专项扩展字段。
3. 每条线索必须明确来源、位置、主体、面积、责任单位、整改时限和销号材料。
4. 对高风险线索实行主要领导盯办、专班核查、限期整改和每周调度。
5. 全部敏感线索仅限内部流转，严禁外传。
