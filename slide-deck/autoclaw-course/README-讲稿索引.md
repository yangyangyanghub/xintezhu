## 关键风险提醒

1. **9 个模型数字**：讲稿里多处引用"9 个模型"，开讲前确认规划院 API 网关当前实际可用模型数
2. **文件访问权限坑**：第 16 页强调"关闭文件访问范围限制"——如果 AutoClaw 最新版改了默认配置，这段可能要更新
3. **演示环节没有逐字稿**：第 17、18 页只有清单和提示，演示具体怎么走需要自己心里有数；建议演示前自己跑一遍，确认每个 demo 都能成功
4. **"百虾大战"梗的接受度**：第一次见的同事可能 get 不到，自我介绍里已经做了铺垫，PPT 第 5 页可以再点一下
