# DESIGN.md - 登录页面设计文档

## 一、页面目标

### 核心目标
- 用户能快速完成账号登录，进入产品主功能
- 降低登录摩擦，减少用户流失

### 成功指标
- 登录表单提交成功率 > 95%
- 登录流程平均耗时 < 30秒
- 登录错误率 < 5%

---

## 二、信息层级

### 层级划分（从高到低）

| 层级 | 内容 | 目的 |
|------|------|------|
| **L1** | 品牌标识 + 页面标题 | 确认当前页面/产品 |
| **L2** | 邮箱输入 + 密码输入 + 登录按钮 | 核心登录操作 |
| **L3** | 记住我 + 忘记密码 | 辅助选项 |
| **L4** | 第三方登录入口 | 备选登录方式 |
| **L5** | 注册入口（如有） | 新用户转化 |

## 四、状态设计

### 输入框状态

| 状态 | 视觉表现 |
|------|----------|
| **默认** | 浅灰边框，占位符文字 |
| **聚焦** | 品牌色边框，输入光标 |
| **输入中** | 内容显示，可清除按钮 |
| **错误** | 红色边框，错误提示文字 |
| **禁用** | 灰色背景，禁止输入 |

### 按钮状态

| 状态 | 视觉表现 |
|------|----------|
| **默认** | 品牌色填充，白色文字 |
| **悬停** | 色彩加深 10% |
| **点击** | 色彩加深 20%，轻微缩放 |
| **加载中** | 加载动画，禁用点击 |
| **禁用** | 灰色填充，不可交互 |

### 复选框状态

| 状态 | 视觉表现 |
|------|----------|
| **未选中** | 空白方框 |
| **选中** | 勾选标记，品牌色边框 |
| **悬停** | 浅色背景 |

## 五、交互规则

### 表单验证

| 字段 | 规则 | 触发时机 |
|------|------|----------|
| 邮箱 | 必填，格式校验 | 失焦时 + 提交时 |
| 密码 | 必填，长度 ≥ 6 | 失焦时 + 提交时 |

### 错误反馈

| 场景 | 提示内容 | 展示位置 |
|------|----------|----------|
| 邮箱为空 | 请输入邮箱地址 | 输入框下方 |
| 邮箱格式错误 | 请输入有效的邮箱地址 | 输入框下方 |
| 密码为空 | 请输入密码 | 输入框下方 |
| 密码错误 | 邮箱或密码错误 | 表单顶部 |
| 账号不存在 | 该账号未注册 | 表单顶部 |
| 网络异常 | 网络异常，请稍后重试 | Toast 提示 |

### 登录流程

```
用户填写表单 → 点击登录 → 前端校验 → 校验失败 → 显示错误提示
                              ↓ 校验成功
                         按钮显示加载态
                              ↓
                         发起登录请求 → 成功 → 存储 Token → 跳转首页
                              ↓ 失败
                         显示错误提示 → 恢复按钮状态
```

### 记住我功能

- **勾选**：登录成功后，Token 有效期延长（如 7 天）
- **不勾选**：Token 有效期较短（如 1 天），浏览器关闭后失效
- **默认状态**：不勾选

### 忘记密码入口

- 点击后跳转到 `/forgot-password` 页面
- 或打开忘记密码弹窗（需确认）

## 六、视觉方向

### 色彩

| 元素 | 颜色 | 说明 |
|------|------|------|
| 主色 | 品牌色 | 登录按钮、聚焦边框、选中态 |
| 背景色 | #FFFFFF / #F5F7FA | 浅色模式主背景 |
| 边框色 | #D9D9D9 | 默认输入框边框 |
| 错误色 | #FF4D4F | 错误提示文字、错误边框 |
| 文字色 | #333333 | 主要文字 |
| 次要文字 | #999999 | 占位符、辅助文字 |

### 字体

| 元素 | 字号 | 字重 |
|------|------|------|
| 页面标题 | 24px | 600 |
| 输入框文字 | 14px | 400 |
| 按钮文字 | 16px | 500 |
| 辅助文字 | 12px | 400 |
| 错误提示 | 12px | 400 |

### 圆角

- 输入框：4px
- 按钮：4px
- 表单容器：8px

### 阴影

- 表单容器：`0 2px 8px rgba(0, 0, 0, 0.1)`（可选）
