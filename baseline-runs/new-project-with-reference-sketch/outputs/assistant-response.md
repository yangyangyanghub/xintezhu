# 首页设计方案

## 一、首屏结构

### 1.1 布局层级
```
┌────────────────────────────────────────────────────────┐
│  导航栏 (固定顶部)                                      │
├────────────────────────────────────────────────────────┤
│                                                        │
│  ┌─────────────────┐    ┌──────────────────────────┐  │
│  │   左侧文案区      │    │    右侧视觉区              │  │
│  │   - 主标题       │    │    - 插图/产品展示         │  │
│  │   - 副标题       │    │                          │  │
│  │   - 双按钮       │    │                          │  │
│  └─────────────────┘    └──────────────────────────┘  │
│                                                        │
├────────────────────────────────────────────────────────┤
│  功能卡片区 (三列等宽)                                   │
│  ┌────────┐  ┌────────┐  ┌────────┐                   │
│  │ 卡片 A  │  │ 卡片 B  │  │ 卡片 C  │                   │
│  └────────┘  └────────┘  └────────┘                   │
└────────────────────────────────────────────────────────┘
```

### 1.2 尺寸规范
| 区域 | 桌面端 (≥1280px) | 平板端 (768-1279px) | 移动端 (<768px) |
|------|-----------------|---------------------|----------------|
| 容器宽度 | 1280px | 100% padding 24px | 100% padding 16px |
| 导航栏高度 | 80px | 72px | 64px |
| 首屏高度 | min-height: 600px | min-height: 500px | 自适应 |
| 卡片间距 | 40px | 24px | 16px |

## 二、内容模块顺序

### 2.1 导航栏模块
1. **Logo 区域** - 左侧，宽度 220px
2. **导航链接** - 中间，横向排列（需确认具体导航项）
3. **CTA 按钮** - 右侧，主按钮样式

### 2.2 Hero 区域模块
**左侧文案区 (占比 ~44%)**
1. 主标题 H1 - 44px/48px，品牌色
2. 副标题 H2 - 24px，灰色辅助文字
3. 按钮组：
   - 主按钮（填充式）
   - 次按钮（描边式）

**右侧视觉区 (占比 ~41%)**
- 主视觉插图/产品展示图（需确认内容形式）

### 2.3 功能卡片模块
三列等宽卡片，每个卡片包含：
- 图标/插图（需确认样式）
- 标题
- 描述文字（需确认内容）

---

## 三、卡片布局详解

### 3.1 布局参数
```
卡片容器: flex 布局，gap: 40px
单卡片宽度: calc((100% - 80px) / 3) ≈ 400px
卡片内边距: 32px
卡片圆角: 20px
卡片背景: #f9fafb
卡片边框: 1px solid #9ca3af
```

### 3.2 响应式行为
| 断点 | 布局 | 说明 |
|------|------|------|
| ≥1024px | 三列横排 | 每列等宽 |
| 768-1023px | 三列横排 | 间距缩小，卡片自适应 |
| <768px | 单列竖排 | 堆叠展示 |

### 3.3 卡片内容结构
```
┌──────────────────────┐
│  [图标 48x48]         │
│  标题 (28px)          │
│  描述文字 (16px)      │
│  可选：链接/按钮       │
└──────────────────────┘
```

## 四、按钮层级

### 4.1 按钮分类
| 层级 | 样式 | 使用场景 |
|------|------|---------|
| **主按钮** | 深色填充 (#111827)，白字，圆角 24px | 核心转化入口 |
| **次按钮** | 白色填充，深色描边 2px，圆角 24px | 辅助操作 |
| **CTA 按钮** | 同主按钮样式 | 导航栏右侧 |

### 4.2 按钮尺寸
| 尺寸 | 高度 | 内边距 | 字号 |
|------|------|--------|------|
| 大 | 48px | 32px | 20px |
| 中 | 36px | 24px | 18px |
| 小 | 32px | 16px | 14px |

### 4.3 交互状态
- **Hover**: 主按钮加深 10%，次按钮背景变灰
- **Active**: 轻微下移 (1px)
- **Disabled**: 透明度 0.5，禁用点击

## 五、响应式行为

### 5.1 断点定义
```css
移动端: max-width: 767px
平板端: min-width: 768px, max-width: 1023px  
桌面端: min-width: 1024px
大屏: min-width: 1280px (最大容器宽度)
```

### 5.2 各模块响应式策略

#### 导航栏
| 断点 | 行为 |
|------|------|
| 桌面端 | Logo + 导航链接 + CTA 水平排列 |
| 平板端 | 导航链接简化或部分隐藏 |
| 移动端 | 汉堡菜单 + Logo + CTA 精简 |

#### Hero 区域
| 断点 | 行为 |
|------|------|
| 桌面端 | 左右两栏布局 |
| 平板端 | 视觉区缩小，文字区占比增大 |
| 移动端 | 单列堆叠，视觉区在上或下（需确认） |

#### 功能卡片
| 断点 | 行为 |
|------|------|
| 桌面端 | 三列横排 |
| 平板端 | 三列横排（紧凑版） |
| 移动端 | 单列堆叠 |

### 5.3 字号缩放
| 元素 | 桌面端 | 平板端 | 移动端 |
|------|--------|--------|--------|
| H1 主标题 | 44px | 36px | 28px |
| H2 副标题 | 24px | 20px | 16px |
| 卡片标题 | 28px | 24px | 20px |
