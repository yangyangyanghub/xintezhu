# 封面图设计

## 文章信息
- 标题：大模型种类全景
- 主题：AI大模型六大类型科普（文本、生图、视频、代码、多模态、语音）
- 风格：科技感、专业、清晰

## 五维参数
- 类型：conceptual（概念隐喻）
- 配色：cool（冷色调：蓝绿青灰）
- 渲染：flat-vector（扁平矢量）
- 文字：title-only（仅标题）
- 氛围：balanced（均衡）

## 视觉设计

### 核心视觉
六个图标卡片代表六种AI模型类型，排列成 2行×3列 或 3行×2列 网格：

1. **文本模型** - 对话气泡 / 文档图标
2. **生图模型** - 图片框架 + 闪光星芒
3. **视频模型** - 播放按钮 / 视频帧
4. **代码模型** - 代码括号 {} / 终端图标
5. **多模态模型** - 眼睛 + 文档组合
6. **语音模型** - 麦克风 / 声波

### 布局
- 左侧60%：图标网格区域
- 右侧40%：标题区域，显示"大模型种类全景"
- 背景：浅灰色 #F8F9FA
- 装饰元素：技术网格线、连接箭头（暗示决策流程）

### 配色方案
- 主色：工程蓝 #2563EB
- 次色：藏青 #1E3A5F
- 强调：青色 #06B6D4
- 点缀：琥珀 #F59E0B

### 扁平矢量风格
- 纯色填充，无渐变
- 圆角矩形卡片
- 简洁几何图形
- 统一描边粗细

### 文字要求
- 标题：大模型种类全景
- 字体：无衬线，现代简洁
- 颜色：深色（#1E3A5F 或 #333333）

## 构图原则
- 留白 40-60%
- 视觉焦点居中或偏左
- 简化图标，无真人面孔
- 图标风格统一

【文字渲染要求】
- 所有中文文字必须清晰可读，笔画完整，无模糊、无乱码、无伪文字
- 文字边缘锐利，呈现印刷级清晰度，彻底消除压缩噪点与边缘溢色
- 字体风格统一，字距适中，排版规整
- 严禁出现无法阅读的乱码字符或残缺笔画
