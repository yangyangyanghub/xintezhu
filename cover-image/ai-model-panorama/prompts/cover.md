Abstract shapes representing core concepts, information hierarchy, clean zones. Use a structured layout that clearly communicates the "6 types classification" concept.

## Visual composition:
- Main visual: Six icon cards representing different AI model types arranged in a 2x3 or 3x2 grid pattern:
  1. Text model: Chat bubble / document icon
  2. Image generation model: Image/photo frame icon with sparkle
  3. Video model: Play button / video frame icon
  4. Code model: Code brackets { } or terminal icon
  5. Multimodal model: Eye + text combined icon (image + document)
  6. Voice model: Microphone / sound wave icon
- Layout: Icons arranged in a clean grid on the left 60% of the image, with title "大模型种类全景" prominently displayed on the right side
- Decorative: Technical grid lines, connecting arrows suggesting a decision flow, geometric accents
