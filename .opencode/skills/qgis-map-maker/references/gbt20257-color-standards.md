| 标准号 | 全称 | 适用范围 |
|--------|------|---------|
| GB/T 20257.1 | 国家基本比例尺地图图式 第1部分：1:500 1:1000 1:2000 | 大比例尺地形图 |
| GB/T 20257.2 | 国家基本比例尺地图图式 第2部分：1:5000 1:10000 | 中比例尺地形图 |
| GB/T 20257.3 | 国家基本比例尺地图图式 第3部分：1:25000 1:50000 1:100000 | 中小比例尺地形图 |

    'color': '#0066CC',

    'capstyle': 'round',

### 地形高程栅格（使用 MapTiler 配色）
```python
from qgis.core import QgsColorRampShader, QgsRasterShader, QgsSingleBandPseudoColorRenderer

# 加载 MapTiler terrain-color-ramp.txt
def load_terrain_ramp(layer):
    ramp_items = []
    with open('references/color-ramps/terrain-color-ramp.txt') as f:
        for line in f:
            if line.startswith('#') or not line.strip():
                continue
            parts = line.strip().split(',')
            if len(parts) >= 5:
                val = float(parts[0])
                color = QColor(int(parts[1]), int(parts[2]), int(parts[3]), int(parts[4]))
                ramp_items.append(QgsColorRampShader.ColorRampItem(val, color, f'{val}m'))
    
    fnc = QgsColorRampShader()
    fnc.setColorRampType(QgsColorRampShader.Interpolated)
    fnc.setColorRampItemList(ramp_items)
    
    shader = QgsRasterShader()
    shader.setRasterShaderFunction(fnc)
    renderer = QgsSingleBandPseudoColorRenderer(layer.dataProvider(), 1, shader)
    layer.setRenderer(renderer)
    layer.triggerRepaint()
```

## 配色记忆口诀

> **蓝水绿树红居民，黄道棕等高，注记分色记心中。**

- 水 = 蓝 | 树 = 绿 | 居民 = 棕红
- 道路 = 黄橙分级 | 等高 = 棕褐
- 注记按要素继承主体色
