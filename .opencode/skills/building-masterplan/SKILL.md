# Skill: 总平面图生成器

建筑总平面图生成，用于展示建筑布局、环境关系和功能分区。

## 工具依赖

- **FreeCAD CLI**：CAD图纸生成
- **阿里千问VL**：彩色总平面渲染

## 触发场景

当用户提到以下关键词时使用此技能：
- 总平面图、总图、场地规划
- 建筑布局、环境关系、功能分区
- 场地设计、规划设计

## 核心指令

### 步骤1：场地分析

输入：项目描述 + 场地条件

分析维度：
- 场地尺寸（长×宽）
- 朝向（南北东西）
- 周边环境（道路、建筑、景观）
- 限制条件（退线、限高、容积率）
- 入口方向（人行、车行）

### 步骤2：布局推导

根据功能需求推导建筑布局：

| 功能类型 | 布局策略 |
|---------|---------|
| 住宅 | 南北通透、间距满足日照 |
| 商业 | 沿街展示、人车分流 |
| 办公 | 集中布局、高效动线 |
| 文化 | 序列空间、仪式感 |

**布局原则**：
- 日照优先：主要功能朝南
- 流线清晰：人车分流、洁污分流
- 景观利用：借景、造景
- 消防合规：登高面、环形通道

## 约束条件

### 必须遵守
- 符合规划规范（退线、间距、日照）
- 消防通道合规（宽度、转弯半径）
- 功能分区合理、流线清晰
- 标注完整（尺寸、图例、指北针）

### 禁止行为
- 建筑间距不满足日照要求
- 消防通道被占用或中断
- 功能分区混乱、流线交叉
- 遗漏关键标注信息

## 质量检查

### 自检清单
- [ ] 建筑布局满足日照间距
- [ ] 消防通道宽度≥4米，转弯半径≥12米
- [ ] 人车分流，流线清晰
- [ ] 退线距离符合规划要求
- [ ] 标注完整（尺寸、图例、指北针、比例尺）
- [ ] 技术指标计算正确
- [ ] DWG文件可正常打开

## 错误处理

| 错误类型 | 处理方式 |
|---------|---------|
| 场地尺寸不明确 | 请用户提供红线图 |
| 布局不合理 | 调整建筑位置重试 |
| 指标超标 | 优化布局方案 |
| CAD生成失败 | 降级为示意图 |
