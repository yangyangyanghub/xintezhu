# 资料索要清单与对接函生成 Prompt

## 任务
你是资深规划项目负责人，请为【____亩 ____性质 地块】的现场调研，生成全套资料索要清单和各单位对接函模板。

## 地块信息（用户填写）
- 地块面积：____ 亩
- 用地性质：____
- 地块位置：____ 市 ____ 区 ____ 街道/镇
- 预计调研时间：____ 年 ____ 月 ____ 日

## 输出要求

### 一、分单位资料索要清单

#### 1. 自然资源和规划局（重点！）
```
需要索要的资料清单：
□ 最新控制性详细规划文本 + 图则（电子版 + 纸质盖章版）
□ 地块红线CAD文件（带坐标）
□ 土地利用总体规划图
□ 城市设计导则（如有）
□ 市政专项规划（给水、排水、电力、燃气、供热）
□ 交通专项规划（道路网、轨道交通、公交）
□ 绿地系统规划
□ 历史文化名城保护规划（如涉及）
□ 征地拆迁相关文件（如有）
□ 国有土地使用权出让合同（如已出让）
```

#### 2. 市政各公司
```
【自来水公司】
□ 周边给水管道平面图（管径、材质、埋深、标高）
□ 最近接水口位置、管径、水压
□ 供水能力证明

【排水公司/住建局】
□ 周边雨水、污水管道平面图
□ 污水处理厂位置、处理能力、余量
□ 雨水排出口位置、标高
□ 防洪标准与洪水位标高

【电力公司】
□ 周边变电站位置、容量、电压等级
□ 高压线路走向图、杆位坐标
□ 最近供电接口位置、容量
□ 地块内现状电力线路图

【燃气公司】
□ 周边燃气管道平面图（管径、压力、材质）
□ 最近接气口位置
□ 供气能力证明

【通信运营商（移动/联通/电信/广电）】
□ 周边通信管道走向图
□ 交接箱位置、容量
```

#### 3. 街道办事处/村委会
```
□ 地块权属证明材料
□ 征地拆迁进展情况
□ 村民安置方案
□ 地块内坟墓、宗祠等特殊设施位置与权属
□ 当地村民诉求与意见
□ 周边社区人口、年龄结构等基本情况
```

#### 4. 其他相关单位
```
【交通局/交警大队】
□ 周边道路网规划图
□ 交叉口渠化设计图
□ 交通流量统计数据
□ 地块开口审批要求

【环保局】
□ 地块环境评估报告（如有）
□ 周边污染源分布
□ 生态红线分布图

【水务局/水利局】
□ 河道蓝线图
□ 防洪规划
□ 地下水位资料

【文物局】
□ 文物保护单位分布图、保护范围
□ 考古调查报告（如有）
```

### 二、正式对接函模板（给每个单位）

```
【单位名称】：

您好！

我单位【____公司/设计院名称】正在开展【____项目名称】的修建性详细规划编制工作，该项目位于【____位置】，用地面积约【____】亩。

为确保规划成果科学、合理、可落地，需要贵单位提供以下资料作为设计依据：

（此处插入上述对应单位的资料清单）

以上资料如需付费，请告知收费标准，我们将按规定办理。如有电子版本，请发送至邮箱：【____】。纸质版请寄送至：【____地址】，联系人：【____】，电话：【____】。

恳请贵单位予以支持配合，万分感谢！

顺颂商祺！

【____公司/设计院名称】（盖章）
____年____月____日

附件：项目范围图
```

### 三、风险预判清单与应对话术

**常见问题与应对策略**：

| 可能遇到的问题 | 应对话术/解决方案 |
|--------------|-----------------|
| "这个资料涉密，不能提供" | "理解理解。那能否请贵单位派专人现场指认一下关键管线位置？我们保证不对外传播，仅用于项目设计参考。" |
| "资料还在更新，暂时没有" | "那能否提供上一版的？我们会标注为过程稿，不做最终依据使用。或者您方便的时候口头告知一下大致走向也可以。" |
| "你们去找甲方要，我们不直接对接" | "好的，我们马上联系甲方协调。请问如果甲方发函过来，大概多久能拿到？我们项目时间比较紧。" |
| "这个不归我们管，你去找XX部门" | "好的，谢谢您的指引。请问XX部门具体对接人您有推荐吗？或者能否帮我们转达一下需求？" |

### 四、给甲方的资料需求清单

```
甲方需要提供的资料：
1. 项目立项/备案文件
2. 土地出让合同或土地权属证明
3. 用地规划许可证（如已办理）
4. 之前做过的可研、概念方案等
5. 甲方对项目的初步设想与要求
6. 项目投资估算与开发节奏
7. 甲方对接人联系方式
8. 与政府各部门的对接渠道资源
```
