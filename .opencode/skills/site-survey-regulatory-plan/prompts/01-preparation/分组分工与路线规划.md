# 调研分组分工与路线规划 Prompt

## 任务
请为【____】地块的现场调研，制定详细的人员分组分工、每日调研路线、时间安排和工具清单。

## 地块基本信息（用户填写）
- 地块面积：____ 亩
- 地块位置：____
- 地块形状特点：____（如：狭长/方正/有河流穿过/有山体等）
- 周边特殊情况：____（如：有高压走廊/有村庄/有工厂/有河流等）

## 输出要求

### 一、推荐分组方案

**【方案A：标准6人团队】（推荐，适合100-300亩地块）**

| 组别 | 人数 | 负责人 | 核心职责 |
|------|------|--------|---------|

| 建构筑物组 | 1人 |  | 建筑统计、质量评估、保留/拆除分类 |
| 市政管线组 | 2人 |  | 给水、排水、电力、燃气、弱电全专业实测 |
| 交通生态组 | 1人 |  | 道路标高、流量统计、地形水文植被调查 |
| 记录与资料员 | 1人 |  | 照片统一命名、表格汇总、每日简报 |

**【方案B：加强版8人团队】（适合300亩以上复杂地块）**

| 组别 | 人数 | 负责人 | 核心职责 |
|------|------|--------|---------|

### 二、每日调研路线规划

**原则**：按区域分片，不走回头路，每天下午4点开始汇总，5点结束外业。

**【第1天：南部片区】（示例，需要根据实际地块调整）**
```
上午 8:30-9:00   团队集合，分发工具，当天任务交底
上午 9:00-12:00  南部区域边界实测、建构筑物调查
中午 12:00-13:30 午餐、短暂休息
下午 13:30-16:00 南部区域市政管线、交通调查
下午 16:00-17:00 当天数据汇总、照片整理、问题清单梳理
下午 17:00-17:30 短会：问题通报、第二天工作调整
```

**【第2天：北部片区】**
```
上午 9:00-12:00  北部区域边界实测、建构筑物调查
下午 13:30-16:00 北部区域市政管线、生态调查
下午 16:00-17:30 当天汇总 + 中期整体进度评估
```

**【第3天：补测与重点复核】**
```
上午 9:00-12:00  前两天发现问题的补测点、争议点复核
下午 13:30-16:00 重点区域加密测量、关键访谈
下午 16:00-18:00 全体汇总、照片整理归档、初步问题清单
```

### 三、每个小组的详细任务清单

**【土地与权属组】当天任务清单**：
- [ ] 所有红线拐点RTK实测，记录坐标与误差
- [ ] 现场钉桩或喷漆标记边界
- [ ] 每个拐点拍带GPS水印照片（至少2个角度）
- [ ] 标注土地利用现状（耕地/林地/草地/水域/建设用地/闲置地）
- [ ] 记录农用地作物类型、种植周期、年产值（访谈农户）
- [ ] 标注临时工棚、堆场、停车场等临时用地
- [ ] 核实飞地、插花地、代征绿地/道路
- [ ] 访谈村委会/权属人，记录权属争议点

**【建构筑物组】当天任务清单**：
- [ ] 每栋建筑编号、拍照（至少3个角度：立面、入口、铭牌）
- [ ] 记录层数、高度、结构类型（砖混/框架/钢结构）
- [ ] 评估建成年代、建筑面积估算
- [ ] 质量评估：墙体开裂、地基沉降、屋面漏水、消防设施情况
- [ ] 判断保留/拆除建议及理由
- [ ] 标注化工厂、加油站、变电站等特殊建筑
- [ ] 记录围墙、挡土墙、水井、化粪池、垃圾站等附属设施
- [ ] 记录广告牌、电线杆、变压器位置与产权单位

**【市政管线组】当天任务清单**：
- [ ] 每个窨井盖打开实测（管径、材质、管底标高、流向）
- [ ] 每根管线至少测3个点：起点、交叉点、地块接口
- [ ] 记录管线铭牌信息、产权单位电话
- [ ] 拍照：井盖全景+铭牌特写
- [ ] 给水：水压实测（如有条件）、接口位置确认
- [ ] 排水：雨水口标高、污水检查井标高
- [ ] 电力：杆号、电压等级、变压器容量
- [ ] 燃气：调压箱位置、压力等级
- [ ] 标注废弃管线位置与埋深

**【交通生态组】当天任务清单**：
- [ ] 周边主要道路等级、红线宽度、路面材质记录
- [ ] 道路标高实测（至少每个交叉口测3个点）
- [ ] 交叉口形式、信号灯配置记录
- [ ] 高峰小时流量统计（早7:30-8:30，晚17:30-18:30）
- [ ] 公交站点位置、线路、发车频率记录
- [ ] 地铁站距离、出入口位置、步行时间
- [ ] 地块内最高点、最低点、坡度变化点标高实测
- [ ] 水体位置、面积、水深、常水位/洪水位记录
- [ ] 古树名木编号、树龄、胸径、保护级别记录
- [ ] 特殊地形记录：冲沟、滑坡、塌陷、陡坎、湿地

### 五、注意事项与安全要求

1. **安全第一！** 打开井盖必须有人监护，设置警示标志
2. **高压线路保持安全距离**，不要在高压线下长时间停留
3. **进入废弃建筑前**先确认结构安全，不要贸然进入
4. **访谈村民时**态度要友好，说明来意，避免冲突
5. **所有照片必须带GPS水印**，这是成果可追溯的核心保证
6. **每天数据必须当天整理归档**，不要攒到最后一起弄
7. **发现重大问题立即拍照+群里通报**，不要隐瞒
