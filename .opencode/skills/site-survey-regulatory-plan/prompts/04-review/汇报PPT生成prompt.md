# 汇报PPT自动生成 Prompt

## 任务
基于调研报告，生成给甲方/政府领导的汇报PPT大纲和内容要点。

## 目标听众与原则
- **听众**：政府领导、甲方高管（非专业背景，关注核心问题和决策点）
- **原则**：少技术细节，多讲问题、影响、决策建议
- **篇幅**：20-25页，30分钟汇报完

## PPT核心结构（12页核心页）

### 第1页：封面
- 标题：【项目名称】现场调研报告
- 副标题：修建性详细规划编制阶段
- 汇报单位、时间

### 第2页：目录
1. 项目概况与调研工作情况
2. 土地与权属现状
3. 建构筑物现状
4. 市政基础设施现状（核心！）
5. 交通系统现状
6. 自然生态与景观资源
7. 施工条件评估
8. 特殊限制因素（一票否决项，重点！）
9. 存在问题与风险分级（核心！）
10. 下一步工作建议

### 第3页：项目概况
- 【左侧】项目位置图
- 【右侧】核心数据卡片：面积、性质、调研时间、团队、实测点数
- 【底部】调研方法：RTK精准测量 + 无人机航测 + 多部门访谈 + 管线探测

### 第4页：土地与权属现状
- 土地利用现状饼图（耕地/建设用地/其他）
- 核心结论：边界误差、权属分布、拆迁户数
- 重点提示：征地拆迁进度是影响开工的关键变量

### 第5页：建构筑物现状
- 建筑分类统计
- 质量评估分布
- 保留/拆除建议
- 特殊建筑（文保/污染/安全隐患）

### 第6-9页：市政基础设施（分4页）
- **第6页**：给水+排水（接口条件、容量、防洪标准）
- **第7页**：电力+燃气（变电站、线路、迁改需求）
- **第8页**：管线综合问题（交叉冲突点、迁改费用预估）
- **第9页**：市政小结（各专业情况表、总投资、周期预估）

### 第10页：交通系统现状
- 周边道路条件
- 出入口可行性建议
- 公共交通情况

### 第11页：自然生态与施工条件
- 地形坡度分析
- 水文、植被、古树名木
- 施工进场条件、水电接驳、弃土场

### 第12页：特殊限制因素（一票否决项）
- 机场净空、铁路/高速控制线
- 输油输气管道、军事设施
- 生态红线、环保要求

### 第13-14页：问题与风险分级（核心！用2页）
- 🔴 一级风险清单（1-2个问题一页，讲清楚影响、费用、建议）
- 🟠 二级风险清单
- 🟡 三级风险清单

### 第15页：下一步工作建议（决策页！）
```
【甲方立即开展】
1. ____
2. ____
3. ____

【设计院立即开展】
1. ____
2. ____

【需政府协调】
1. ____
2. ____

【时间节点建议】
• 第1-2周：完成____
• 第3-4周：完成____
• 第2个月：完成____
```

### 第16页：汇报结束
- 感谢聆听
- 联系人、电话、邮箱

## 汇报技巧提示

### 1. 一页只讲一个核心观点
不要一页塞太多内容，一页讲清楚一个点就好。

### 2. 多用图表、少用文字
- 能用饼图/柱状图的不用表格
- 能用表格的不用大段文字
- 关键数据放大、标红

### 3. 领导最关心的3个问题（一定要讲清楚）
1. **要花多少钱？**（市政配套、拆迁、迁改等）
2. **要花多长时间？**（前期手续、施工周期等）
3. **有什么重大风险？**（哪些可能导致项目干不成或大幅超支）

### 4. 风险分级的汇报技巧
- **🔴红色**：翻页时停顿2秒，语气凝重，"这个问题非常关键，我建议..."
- **🟠橙色**：正常语速，"这个问题需要重视..."
- **🟡黄色**：快速带过，"这些是设计阶段需要考虑的细节问题"

### 5. 政府领导汇报注意事项
- 先讲结论，再讲过程
- 先讲问题，再讲成绩
- 多提"需要领导协调解决的事项"
- 少讲技术细节，多讲决策建议
- 留下书面报告，方便领导后续查阅
