# Skill: 设计概念图生成器

建筑设计概念图生成，用于表达设计理念、风格方向和空间氛围。

## 工具依赖

- **阿里千问VL**：图片理解 + 概念图生成

## 触发场景

当用户提到以下关键词时使用此技能：
- 概念图、概念设计、设计理念
- 风格方向、空间氛围、意向图
- 建筑概念、方案概念

## 核心指令

### 步骤1：需求解析

输入：项目描述（文字/图片/草图）

解析维度：
- 建筑类型（住宅/商业/办公/文化）
- 设计风格（现代/古典/极简/地域/参数化）
- 空间氛围（温馨/庄重/活力/静谧）
- 材质偏好（玻璃/石材/木材/金属/混凝土）
- 环境关系（城市/自然/滨水/山地）

### 步骤2：概念提炼

从需求中提炼核心概念词：

| 维度 | 概念词示例 |
|------|-----------|
| 形态 | 流动、折叠、堆叠、悬挑、围合 |
| 空间 | 通透、层次、序列、渗透、留白 |
| 材质 | 温润、粗粝、精致、轻盈、厚重 |
| 氛围 | 静谧、活力、庄重、亲切、神秘 |

### 步骤3：提示词构建

**基础模板**：
```
建筑概念图，{风格}风格，
设计理念：{核心概念}，
空间氛围：{氛围关键词}，
材质表现：{材质描述}，
{光照条件}，{视角}，
高质量，建筑设计表现，概念草图风格
```

**风格变体**：

| 风格 | 提示词关键词 |
|------|-------------|
| 概念草图 | sketch style, hand drawing, concept |
| 意向拼贴 | collage, mood board, inspiration |
| 水墨风 | ink wash, chinese painting, 水墨 |
| 油画风 | oil painting, impressionist, 油画 |
| 赛博朋克 | cyberpunk, neon, futuristic |
| 地域特色 | {地域} architecture, local style |

### 步骤4：生成与筛选

1. 调用千问生图API，生成3-5张候选
2. 根据概念一致性筛选
3. 选择最符合设计理念的2-3张

## 输入格式

```
项目描述：
- 建筑类型：住宅/商业/办公/文化
- 设计风格：现代/古典/极简/地域
- 空间氛围：温馨/庄重/活力/静谧
- 材质偏好：玻璃/石材/木材/金属
- 核心概念：（一句话设计理念）

参考图片（可选）：
- 意向图
- 草图
- 场地照片

输出要求：
- 风格偏好：概念草图/意向拼贴/艺术风格
- 数量：2-3张
```

## 约束条件

### 必须遵守
- 概念图需体现设计理念
- 风格方向明确、一致
- 材质表现真实可信
- 空间氛围符合定位

### 禁止行为
- 概念模糊、风格混乱
- 过度写实（概念图应有想象空间）
- 忽略场地环境关系
- 使用与项目不符的风格

## 质量检查

### 自检清单
- [ ] 概念图体现了设计理念
- [ ] 风格方向明确、一致
- [ ] 材质表现真实可信
- [ ] 空间氛围符合定位
- [ ] 图片清晰、构图合理
- [ ] 有2-3张不同角度/氛围

## 错误处理

| 错误类型 | 处理方式 |
|---------|---------|
| 需求不明确 | 请用户补充设计理念 |
| 风格不匹配 | 调整提示词重新生成 |
| 效果不理想 | 增加参考图或细化描述 |
| 生成失败 | 降低复杂度重试 |

## 使用示例

### 示例1：现代住宅

**输入**：
```
项目：现代住宅
风格：极简现代
氛围：静谧、自然
材质：白色涂料、玻璃
概念：与自然对话
```

**输出**：
```
[概念图1]
设计理念：与自然对话
风格方向：极简现代，白色体量
材质表现：白色涂料+玻璃幕墙
空间氛围：通透、轻盈、静谧

[概念图2]
设计理念：与自然对话
风格方向：庭院空间，室内外融合
材质表现：木材+玻璃
空间氛围：温暖、自然、层次

概念词总结：
- 形态：简洁、纯粹
- 空间：通透、渗透
- 材质：温润、轻盈
- 氛围：静谧、自然
```

### 示例2：文化建筑

项目：文化艺术中心

氛围：庄重、文化感

概念：传统与现代的对话

```

风格方向：现代地域，抽象传统元素
材质表现：石材+金属线条

设计理念：传统与现代的对话

材质表现：混凝土+木材
空间氛围：层次、序列、光影

概念词总结：
- 形态：抽象、隐喻
- 空间：序列、仪式
- 材质：厚重、精致
- 氛围：庄重、文化
```
