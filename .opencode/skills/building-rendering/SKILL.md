# Skill: 效果图生成器

建筑效果图生成，用于展示建筑外观、材质表现和空间氛围。

## 工具依赖

- **阿里千问VL**：效果图生成

## 触发场景

当用户提到以下关键词时使用此技能：
- 效果图、建筑表现、建筑渲染
- 外观图、透视图、鸟瞰图
- 夜景图、日景图、黄昏图
- 建筑可视化、建筑表现图

## 核心指令

### 步骤1：视角选择

| 视角 | 适用场景 | 特点 |
|------|---------|------|
| 透视图 | 方案展示 | 最直观，有空间感 |
| 鸟瞰图 | 总体关系 | 展示整体布局 |
| 人视图 | 空间感受 | 有代入感 |
| 仰视图 | 建筑气势 | 突出建筑高度 |
| 室内图 | 空间体验 | 展示内部空间 |

**视角选择建议**：
- 方案汇报：透视图 + 鸟瞰图
- 客户沟通：人视图 + 室内图
- 投标展示：多角度组合

### 步骤2：光照设计

| 光照 | 氛围 | 提示词 |
|------|------|--------|
| 日景 | 清晰、明亮 | daylight, sunny, bright |
| 黄昏 | 温暖、浪漫 | sunset, golden hour, warm |
| 夜景 | 神秘、氛围 | night, evening, illuminated |
| 阴天 | 柔和、均匀 | overcast, soft light |
| 晨曦 | 清新、希望 | dawn, morning, fresh |

**光照组合**：
- 商业建筑：黄昏 + 夜景（暖光内透）
- 住宅建筑：日景 + 黄昏（温馨感）
- 文化建筑：日景 + 夜景（庄重感）
- 办公建筑：日景（专业感）

### 步骤3：材质表现

| 材质 | 质感 | 提示词关键词 |
|------|------|-------------|
| 玻璃 | 通透、反射 | glass, reflective, transparent |
| 石材 | 厚重、稳重 | stone, solid, massive |
| 金属 | 精致、现代 | metal, steel, aluminum |
| 木材 | 温暖、自然 | wood, timber, warm |
| 混凝土 | 粗粝、力量 | concrete, brutalist |
| 涂料 | 简洁、纯粹 | white, painted, clean |

**材质搭配**：
- 现代风格：玻璃 + 金属 + 白色涂料
- 地域风格：石材 + 木材 + 混凝土
- 极简风格：白色涂料 + 玻璃
- 工业风格：混凝土 + 金属 + 玻璃

### 步骤4：环境设计

**环境要素**：
- 天空：晴天/多云/黄昏/夜景
- 植物：乔木/灌木/草坪
- 水景：水池/喷泉/水面
- 人物：活动人群
- 车辆：汽车/自行车
- 道路：车行道/人行道

**环境氛围**：
- 商业：繁华、活力、人多
- 住宅：安静、舒适、绿化好
- 文化：庄重、文化感、艺术性
- 办公：专业、高效、现代

### 步骤6：生成与筛选

1. 调用千问生图API，生成5-8张候选
2. 根据以下标准筛选：
   - 建筑形态准确
   - 材质表现真实
   - 光照氛围符合
   - 环境协调统一
3. 选择3-5张最佳效果图

## 输入格式

```
项目描述：
- 建筑类型：住宅/商业/办公/文化
- 建筑风格：现代/古典/极简/地域
- 建筑规模：X层，高度X米
- 材质构成：玻璃X%、石材X%、金属X%

效果图需求：
- 视角：透视/鸟瞰/人视/夜景
- 光照：日景/黄昏/夜景
- 环境：商业/住宅/文化/办公

- 意向图
- 材质参考
- 风格参考

输出要求：
- 分辨率：4K/8K
- 风格：写实/概念/艺术
```

## 约束条件

### 必须遵守
- 建筑形态准确、比例正确
- 材质表现真实、质感可信
- 光照氛围符合设计意图
- 环境协调、风格统一

### 禁止行为
- 建筑形态变形、比例失调
- 材质失真、质感虚假
- 光照混乱、氛围不符
- 环境杂乱、风格不统一

## 质量检查

### 自检清单
- [ ] 建筑形态准确、比例正确
- [ ] 材质表现真实、质感可信
- [ ] 光照氛围符合设计意图
- [ ] 环境协调、风格统一
- [ ] 图片清晰、构图合理
- [ ] 有3-5张不同视角/光照
- [ ] 分辨率满足要求

## 错误处理

| 错误类型 | 处理方式 |
|---------|---------|
| 建筑形态不准确 | 调整提示词、增加参考图 |
| 材质表现失真 | 细化材质描述 |
| 光照氛围不符 | 调整光照描述 |
| 环境杂乱 | 简化环境描述 |
