# Skill: 分析图生成器

建筑分析图生成，用于展示功能分区、流线组织、日照分析等设计逻辑。

## 工具依赖

- **阿里千问VL**：分析图生成
- **FreeCAD CLI**：辅助标注

## 触发场景

当用户提到以下关键词时使用此技能：
- 分析图、功能分析、流线分析
- 日照分析、风环境分析
- 交通分析、景观分析
- 设计分析、方案分析

## 核心指令

### 步骤1：分析类型识别

输入：项目描述 + 分析需求

分析类型：

| 类型 | 内容 | 用途 |
|------|------|------|
| 功能分区 | 公共/半公共/私密分区 | 功能合理性 |
| 流线分析 | 人流/车流/物流 | 动线效率 |
| 日照分析 | 采光/遮阳/阴影 | 规范合规 |
| 景观分析 | 视线/对景/借景 | 景观价值 |
| 交通分析 | 入口/道路/停车 | 交通组织 |
| 风环境 | 通风/防风/舒适度 | 环境品质 |
| 消防分析 | 登高面/救援通道 | 消防合规 |

### 步骤2：数据准备

根据分析类型准备数据：

**功能分区**：
- 各功能区位置
- 各功能区面积
- 功能关系矩阵

**流线分析**：
- 入口位置
- 流线路径
- 流线交叉点

**日照分析**：
- 建筑朝向
- 建筑高度
- 日照标准（大寒日/冬至日）

**景观分析**：
- 景观资源位置
- 视线通廊
- 对景关系

### 步骤3：提示词构建

**功能分区图**：
```
建筑功能分区图，
{建筑类型}建筑群，
功能分区：{分区描述}，
彩色色块标注，
图例说明，
规划设计图风格，鸟瞰视角
```

**流线分析图**：
```
建筑流线分析图，
{建筑类型}建筑群，
流线类型：{人流/车流/物流}，
箭头标注流线方向，
彩色流线，
图例说明，规划设计图风格
```

**日照分析图**：
```
建筑日照分析图，
{建筑类型}建筑群，
日照时间：{X小时}，
阴影范围标注，
采光分析，
图例说明，规划设计图风格
```

**景观分析图**：
```
建筑景观分析图，
{建筑类型}建筑群，
景观资源：{位置描述}，
视线通廊标注，
对景关系，
图例说明，规划设计图风格
```

**交通分析图**：
```
建筑交通分析图，
{建筑类型}建筑群，
入口位置：{位置描述}，
道路系统：{道路描述}，
停车区域：{停车描述}，
图例说明，规划设计图风格
```

**消防分析图**：
```
建筑消防分析图，
{建筑类型}建筑群，
消防通道：{通道描述}，
登高面：{登高面位置}，
救援场地：{场地位置}，
图例说明，规划设计图风格
```

### 步骤4：图例设计

**色彩规范**：

| 分析类型 | 主色 | 辅助色 |
|---------|------|--------|
| 功能分区 | 蓝/绿/黄/橙 | 灰/白 |
| 流线分析 | 红/蓝/绿 | 灰/白 |
| 日照分析 | 黄/橙/红 | 灰/白 |
| 景观分析 | 绿/蓝 | 灰/白 |
| 交通分析 | 灰/蓝 | 绿/白 |
| 消防分析 | 红/橙 | 灰/白 |

**图例元素**：
- 色块：功能分区
- 箭头：流线方向
- 虚线：视线通廊
- 数字：日照时长
- 图标：入口/节点

### 步骤5：生成与标注

1. 调用千问生图API生成基础图
2. 添加分析标注
3. 添加图例说明
4. 添加指北针和比例尺

## 输入格式

```
项目描述：
- 建筑类型：住宅/商业/办公/文化
- 建筑规模：X栋，每栋X层
- 功能构成：商业X%、住宅X%、配套X%

分析需求：
- 分析类型：功能分区/流线/日照/景观/交通/消防
- 分析重点：（重点关注什么）
- 输出形式：单独图/组合图

场地条件：
- 场地尺寸：X米 × X米
- 朝向：主朝向
- 周边环境：道路、景观资源

输出要求：
- 图纸比例：1:500/1:1000
- 输出格式：PNG
- 图例要求：详细/简洁
```

## 约束条件

### 必须遵守
- 分析逻辑清晰、数据准确
- 图例明确、色彩规范
- 标注完整、信息无遗漏
- 符合相关规范要求

### 禁止行为
- 分析逻辑混乱、数据矛盾
- 图例不清、色彩混乱
- 遗漏关键分析内容
- 不符合规范要求

## 质量检查

### 自检清单
- [ ] 分析类型选择正确
- [ ] 分析数据准确无误
- [ ] 图例清晰、色彩规范
- [ ] 标注完整、信息无遗漏
- [ ] 分析逻辑清晰、结论明确
- [ ] 符合相关规范要求
- [ ] 图片清晰、构图合理

## 错误处理

| 错误类型 | 处理方式 |
|---------|---------|
| 分析类型不明确 | 请用户确认分析需求 |
| 数据不完整 | 请用户补充数据 |
| 分析逻辑矛盾 | 检查数据、调整分析 |
| 生成效果差 | 调整提示词重试 |

## 使用示例

### 示例1：住宅小区功能分区

**输入**：
```
项目：住宅小区
规模：5栋18层住宅
功能：住宅+底商+配套
分析类型：功能分区
```

**输出**：
```
[功能分区图]
分区说明：
- 商业区：沿街底商，面积5000㎡
- 住宅区：5栋住宅，面积80000㎡
- 配套区：会所+幼儿园，面积3000㎡
- 景观区：中心花园，面积5000㎡

图例：
- 橙色：商业区
- 蓝色：住宅区
- 黄色：配套区
- 绿色：景观区

文件：analysis_function.png
```
