# 现场调研空白表格模板说明

## 使用说明
以下为各类调研表格的标准格式，可直接复制到Excel中使用。所有表格在现场使用时，必须同时拍摄对应照片，照片编号与表格序号一致。

---

## 表格一：用地边界实测记录表

| 序号 | 点号 | X坐标(m) | Y坐标(m) | Z坐标(m) | 与控规红线误差(m) | 位置描述 | 照片编号 | 备注 |
|------|------|----------|----------|----------|------------------|---------|---------|------|
| 1 | B-01 | | | | | | | |
| 2 | B-02 | | | | | | | |
| 3 | B-03 | | | | | | | |
| ... | | | | | | | | |

**填写说明**：
- 误差≤0.5m为合格，>0.5m需重测并注明原因
- 位置描述要具体到"东北角拐点""与XX路交叉口西50米"等
- 每个点至少拍2张照片：1张特写+1张全景

## 表格二：建构筑物调查表

| 序号 | 建筑编号 | 建筑名称/地址 | 层数 | 高度(m) | 结构类型 | 建成年代 | 建筑面积(㎡) | 质量等级 | 保留/拆除建议 | 照片编号 | 备注 |
|------|---------|--------------|------|---------|---------|---------|------------|---------|-------------|---------|------|
| 1 | JZ-01 | | | | 砖混/框架/钢结构 | | | A/B/C/D级 | 保留/拆除 | | |
| 2 | JZ-02 | | | | | | | | | | |
| ... | | | | | | | | | | | |

**质量等级说明**：
- A级（完好）：建成10年以内，结构完好，无明显裂缝沉降
- B级（一般）：建成10-20年，结构基本完好，局部裂缝
- C级（较差）：建成20年以上，明显裂缝沉降
- D级（危房）：结构严重损坏，有安全隐患

**填写说明**：
- 每栋建筑至少拍3张照片：正立面、侧立面、内部典型空间
- 特殊建筑（化工厂、加油站、变电站）需在备注栏注明
- 围墙、挡土墙、水井等附属设施单独记录

**管线类型代码**：
- G=给水，W=污水，Y=雨水，E=电力，R=燃气，T=通信

**填写说明**：

- 管径如果看不清，用卷尺量井内壁估算
- 有铭牌的拍铭牌特写照片

**访谈记录要点**：
1. 先介绍身份和来意，取得对方信任
2. 关键数据要反复确认，不要怕追问
3. 有争议的地方要原话记录，不要加工
4. 能录音尽量录音，事后整理成文字

**每日晚上必须做**：
1. 所有组员把当天发现的问题汇总到这张表
