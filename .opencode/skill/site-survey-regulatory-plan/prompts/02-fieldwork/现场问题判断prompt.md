# 现场问题AI辅助判断 Prompt

## 任务
现场调研时发现问题，AI立即辅助判断严重等级、影响范围和应对建议。

## 输入格式
用户输入：
- 问题描述：____（越详细越好）
- 问题位置：____（最好有GPS坐标）
- 现场照片：____（如有，描述照片内容）
- 初步判断：____（调研人员的第一感觉）

## AI输出格式

### 一、严重等级判定
```
🔴 一级风险（一票否决/重大安全隐患）
   → 影响：可能导致项目选址变更、重大方案调整、重大安全事故
   → 应对：立即停止相关区域工作，第一时间上报甲方+政府部门

🟠 二级风险（重大影响）
   → 影响：可能导致设计方案重大调整、增加工程造价10%以上、工期延误1个月以上
   → 应对：详细测量记录、专项调研、设计阶段重点考虑

🟡 三级风险（一般影响）
   → 影响：局部设计调整、工程造价增加有限、不影响整体方案
   → 应对：准确记录位置和参数、正常设计时考虑即可
```

### 二、需要补充测量/调查的信息清单
```
[ ] 精确坐标与标高（RTK实测）
[ ] 影响范围测量（长、宽、高、深度）
[ ] 周边参照物距离测量
[ ] 不同角度照片（至少3张）
[ ] 视频记录（动态过程，如排水情况）
[ ] 权属单位访谈（产权人、管理单位）
[ ] 周边居民访谈（历史情况、影响）
[ ] 查阅相关规范标准
[ ] 查阅相关规划文件
```

### 三、对后续规划设计的可能影响分析
```
1. 对总平面布局的影响：
   - 是否需要避让？避让距离多少？
   - 是否影响建筑出入口、道路布局？

2. 对指标核算的影响：
   - 是否占用建设用地？
   - 是否影响容积率、建筑密度、绿地率？
   - 是否需要配套特殊设施？

3. 对市政管线的影响：
   - 是否影响管线走向？
   - 是否需要迁改？迁改费用大概多少？
   - 是否影响接口位置？

4. 对交通组织的影响：
   - 是否影响出入口设置？
   - 是否影响道路标高？
   - 是否影响停车布局？

5. 对生态环境的影响：
   - 是否涉及生态保护？
   - 是否需要环评专项论证？
```

### 四、相关规范标准提示
```
自动匹配相关规范条文：
- 《城市用地分类与规划建设用地标准》GB50137-2011
- 《城市居住区规划设计标准》GB50180-2018
- 《建筑设计防火规范》GB50016-2014（2018年版）
- 《城市工程管线综合规划规范》GB50289-2016
- 《防洪标准》GB50201-2014
- 其他相关专项规范

提示：具体条文内容需要人工查阅确认，AI仅提供方向参考。
```

### 五、典型问题快速判断模板

#### 【问题类型A：发现高压线/高压走廊】
```
严重等级：🟠 二级风险（影响建筑布局，可能需要退距）

关键测量项：
- 电压等级（多少kV？）
- 杆/塔坐标、高度
- 线路走向、与地块红线的距离
- 高压走廊宽度要求（按规范）

设计影响：
- 高压走廊内不能有永久性建筑
- 走廊外多少距离内建筑有限制（不同电压等级不同）
- 是否影响地块出入口
- 是否需要迁改？迁改费用大概多少、难度如何
```

#### 【问题类型B：发现河道/防洪问题】
```
严重等级：🔴/🟠 根据防洪标准判断

关键测量项：
- 常水位、历史最高洪水位
- 河道蓝线范围
- 堤防等级、堤顶标高
- 地块最低点标高与洪水位高差

设计影响：
- 是否满足防洪标准？
- 是否需要抬高场地？造价增加多少？
- 蓝线内不能建设
- 排水出口标高是否满足？
```

#### 【问题类型C：发现污染企业/垃圾站/化工厂】
```
严重等级：🟠/🔴 根据污染程度判断

关键调查项：
- 企业类型、主要污染物
- 是否有环保审批、是否达标排放
- 与地块的距离、主导风向下风向影响
- 是否有土壤、地下水污染历史
- 是否有搬迁计划

设计影响：
- 是否需要环境评估？
- 卫生防护距离是否满足？
- 是否影响居住/商业价值？
- 是否需要防污染措施？
```

#### 【问题类型D：发现文物/历史建筑】
```
严重等级：🔴 一级风险（文物保护有严格法律规定）

关键调查项：
- 文物等级（全国重点/省级/市级/一般）
- 保护范围、建设控制地带范围
- 保护规划要求
- 文物局意见

设计影响：
- 保护范围内原则上不能动土
- 建设控制地带内建设需要文物部门审批
- 设计风格需要与文物协调
- 可能影响整体方案布局
```

#### 【问题类型E：发现地质问题（滑坡/塌陷/软土）】
```
严重等级：🟠 二级风险

关键调查项：
- 影响范围、深度
- 历史发生情况、频率
- 地质勘察报告建议
- 周边类似地质情况的处理方式

设计影响：
- 是否需要地基处理？费用增加多少？
- 是否影响建筑布局？
- 是否需要专项地质勘察？
```

## 使用提示
1. 不要完全依赖AI判断，最终结论需要专业工程师确认
2. 凡是拿不准的，一律按高一级风险处理，宁可信其有不可信其无
3. 所有问题必须有照片、坐标、文字记录，做到可追溯
4. 重大问题当天就要向甲方和项目负责人汇报
