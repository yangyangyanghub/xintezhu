# 统计表格自动生成 Prompt

## 任务
基于调研原始数据，自动生成各类标准化统计表格。

## 输入数据
- 建构筑物调查表原始数据
- 市政管线实测表原始数据
- 问题汇总表原始数据
- 其他相关调研数据

## 输出要求：四个核心表格

---

**格式要求**：

| 序号 | 建筑类型 | 结构类型 | 栋数 | 单栋平均面积(㎡) | 总面积(㎡) | 拆除难度评估 | 拆除费用估算(万元) | 备注 |

| 1 | 住宅 | 砖混 | | | | 中 | | |
| 2 | 住宅 | 框架 | | | | 中 | | |
| 3 | 厂房 | 钢结构 | | | | 低 | | |
| 4 | 厂房 | 砖混 | | | | 中 | | |
| 5 | 商业 | 框架 | | | | 中 | | |
| 6 | 简易棚屋 | 砖木/彩钢 | | | | 低 | | |
| **合计** | - | - | | - | | - | | |

**拆除难度评估标准**：
- 低：单层、简易结构、无大型设备
- 中：2-6层、常规结构、周边有一定空间
- 高：7层以上、周边密集、有污染/危险品、有特殊设备

**拆除费用估算参考**：
- 砖混：80-120元/㎡
- 框架：100-150元/㎡
- 钢结构：50-100元/㎡（视是否回收钢材）
- 简易棚屋：30-50元/㎡

### 表格二：保留建筑质量评估表

**格式要求**：

| 序号 | 建筑编号 | 建筑名称 | 层数 | 结构 | 建成年代 | 面积(㎡) | 质量等级 | 改造费用估算(万元) | 使用建议 | 备注 |

| 1 | | | | | | | A级（完好） | | 可直接利用 | |
| 2 | | | | | | | B级（一般） | | 需局部改造 | |
| 3 | | | | | | | C级（较差） | | 需大修或拆除 | |
| 4 | | | | | | | D级（危房） | | 建议拆除 | |

**质量等级说明**：
- **A级（完好）**：建成10年以内，结构完好，无明显裂缝沉降，水电消防齐全
- **B级（一般）**：建成10-20年，结构基本完好，局部裂缝，设备部分老化
- **C级（较差）**：建成20年以上，结构有明显裂缝、沉降，水电设施陈旧
- **D级（危房）**：结构严重损坏，有安全隐患，建议拆除

**改造费用估算参考**：
- A级：200-300元/㎡（基本装修）
- B级：400-600元/㎡（局部结构加固+装修）
- C级：800-1200元/㎡（全面结构加固+设施更新）

### 表格四：风险分级清单（核心交付物！）

**格式要求（必须按此格式）**：

| 序号 | 风险等级 | 风险描述 | 具体位置 | 影响范围与程度 | 产生原因分析 | 解决建议与措施 | 责任单位 | 紧急程度 | 预计费用(万元) |
|------|---------|---------|---------|-------------|-------------|---------------|---------|---------|--------------|
| 1 | 🔴一级 | | | | | | | 立即处理 | |
| 2 | 🔴一级 | | | | | | | 立即处理 | |
| 3 | 🟠二级 | | | | | | | 尽快处理 | |
| 4 | 🟠二级 | | | | | | | 尽快处理 | |
| 5 | 🟡三级 | | | | | | | 设计考虑 | |
| 6 | 🟡三级 | | | | | | | 设计考虑 | |

**风险等级判定标准**：

**🔴 一级风险（一票否决/重大安全隐患）**
- 可能导致项目选址变更
- 可能导致重大方案调整
- 可能发生重大安全事故
- 可能导致工程造价增加30%以上

**🟠 二级风险（重大影响）**
- 可能导致设计方案重大调整
- 可能导致工程造价增加10%-30%
- 可能导致工期延误1个月以上
- 可能引发周边居民重大投诉

**🟡 三级风险（一般影响）**
- 仅影响局部设计调整
- 工程造价增加10%以内
- 不影响整体方案可行性

## 生成提示

1. 所有表格必须有合计行，数据要准确
2. 费用估算要注明："此为初步估算，实际费用以专业单位报价为准"
3. 备注栏要填写特殊情况说明
4. 表格生成后要做数据一致性检查：分项加总是否等于合计？
5. 费用估算请按当地市场行情调整，本prompt提供的是全国平均参考值
