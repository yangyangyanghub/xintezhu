# CAD现状图AI辅助绘制 Prompt

## 任务
将RTK测点坐标和现场记录数据，AI辅助生成标准CAD现状图。

## 输入数据格式
测点数据Excel表格，至少包含以下列：
- 点号
- X坐标（北京54/CGCS2000等）
- Y坐标
- Z坐标（标高）
- 类型（边界点/建筑角点/井位/控制点/管线点等）
- 备注（管径、建筑层数、材质等）

## 输出要求：分层CAD现状图（1:500或1:1000比例）

## 标注规范（必须统一）

### 建筑标注
```
建筑编号（圆圈内）
层数F
面积㎡

示例：
  ┌─┐
  │3│  B-03
  └─┘  3F  450㎡
```

### 管线标注
```
给水管线：  →  DN300  PE  1.5m  →
              管径  材质  埋深

排水管线：  →  DN600  i=3‰  底标高23.50  →
              管径    坡度    管内底标高

电力管线：  ⚡  10kV  8根  2.0m
                    电缆根数 埋深

燃气管线：  🔥  DN150  0.4MPa  1.8m
                        压力   埋深
```

### 坐标与标高标注
```
坐标标注：
X = 3546789.123
Y = 543210.987

标高标注：
▼ 25.36 （地面标高）
▽ 23.45 （管底标高）
```

## AI绘图步骤

### 第一步：数据导入与检查
1. 将Excel测点数据整理成CSV格式（点号,X,Y,Z,类型,备注）
2. 检查坐标系统一致性（所有点必须同一坐标系）
3. 检查是否有明显错误点（坐标突变、标高突变、重复点号）
4. 删除明显错误的测点并记录

### 第二步：自动连线生成地物
```
【边界线生成】
- 按点号顺序连接边界拐点
- 闭合多边形
- 每个拐点标注X、Y坐标

【建筑轮廓生成】
- 同一建筑的点自动连线闭合
- 建筑居中放置编号圆圈
- 标注层数、面积（面积由CAD自动计算）

【道路生成】
- 先画道路中心线
- 按道路宽度左右偏移生成路边线
- 标注道路名称、等级、宽度
```

### 第三步：管线绘制
```
【给水】
- 蓝色线，带箭头指示流向
- 每隔30-50米标注管径、材质
- 三通、弯头、阀门位置标注

【排水】
- 污水绿色、雨水蓝色
- 每个井标注井底标高
- 标注管径、坡度

【电力】
- 粉色线，带闪电符号
- 标注电压等级、回路数
- 变压器位置用专门符号

【燃气】
- 橙色线，带火焰符号
- 标注管径、压力等级
```

### 第四步：等高线与地形（如有足够测点）
1. 将所有标高测点导入生成三角网
2. 按0.5米或1米等高距生成等高线
3. 首曲线细实线，计曲线粗实线
4. 等高线标高标注在平缓处

### 第五步：图面整饰
1. 添加图名："【项目名称】现状图"
2. 添加图例（所有用到的符号说明）
3. 添加比例尺、指北针
4. 添加坐标系统说明、测量日期
5. 添加"技术说明"栏：
   - 测量依据、测量方法
   - 点位精度说明
   - 数据来源说明
   - 特殊情况备注

## 质量检查清单（AI绘图后必须人工检查）

- [ ] **图层是否正确**：所有地物在对应图层，没有放错层
- [ ] **坐标是否正确**：抽查10%的点，坐标与原始数据一致
- [ ] **标高是否正确**：抽查20%的标高标注与原始数据一致
- [ ] **管线标注是否完整**：管径、材质、标高、坡度都有标注
- [ ] **建筑标注是否完整**：编号、层数、面积都有标注
- [ ] **是否有遗漏测点**：导入的点数量与原始数据一致
- [ ] **是否有重复线段**：同一位置没有画两遍线
- [ ] **图面是否整洁**：标注不重叠、不压盖重要地物

## 常用CAD快捷命令参考（给AI绘图助理）
```
POINT     - 画点
LINE      - 画线
PLINE     - 画多段线
CIRCLE    - 画圆
TEXT/MTEXT - 写文字
DIMLINEAR - 线性标注
LIST      - 查询对象信息
AREA      - 计算面积
ID        - 查询点坐标
PROPERTIES - 打开特性面板（改图层、颜色、线型）
```

**提示**：AI辅助绘图是提高效率的工具，最终图纸质量责任在绘图人，必须经过人工检查和校对才能交付使用。
