# Theme Directory

36 preset themes organized by style family. Each theme defines a cohesive color palette and visual personality for HTML presentations.

> **Note**: All themes use Chinese-first font system — `Noto Serif SC` for titles, `Noto Sans SC` for body. No gradients — solid colors only.

---

## Minimal

Clean, whitespace-heavy themes for maximum clarity. Best for executive summaries, academic reports, and data-focused decks.

| theme-name | style description | best for |
|---|---|---|
| `minimal-white` | Pure white canvas, subtle gray borders, maximum breathing room | Board presentations, clean reports |
| `editorial-serif` | Magazine-quality typography with warm off-white background | Thought leadership, editorial-style decks |
| `japanese-minimal` | Zen-inspired, ultra-restrained palette, asymmetric whitespace | Design portfolios, brand stories |
| `corporate-clean` | Standard corporate gray/white/blue, universally safe | Quarterly reviews, investor updates |
| `academic-paper` | Paper-texture feel, footnote-style accents, conservative | Conference talks, thesis defense |

### Quick Rules

1. **Uncertain? Use `corporate-clean` or `minimal-white`** — these are safe defaults
2. **Never match dark theme for morning meetings** — dark themes require controlled lighting
3. **Code-heavy? Always pick a developer theme** — syntax highlighting depends on the palette
4. **Consistency: apply one theme to entire deck** — mixing themes causes visual chaos
