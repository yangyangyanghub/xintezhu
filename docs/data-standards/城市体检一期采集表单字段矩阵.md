# 城市体检一期采集表单字段矩阵

## 1. 用途

本文档补齐 [[城市体检一期采集表单设计]] 中“模板级设计”到“可配置字段 schema”的缺口，用于指导动态表单配置、字段映射、质检规则挂接和入库实现。

字段矩阵覆盖：

1. 通用底座字段
2. 空间采集字段
3. 矢量任务下发字段
4. 指标填报字段
5. 数据源登记字段
6. 照片附件字段
7. 审核质检字段
8. 专项体检表单字段
9. 城市更新资源子类字段
10. 表单与质检规则映射

## 2. 字段 schema 标准

每个可配置字段至少包含以下属性。

| 属性 | 含义 | 示例 |
|---|---|---|
| formCode | 表单编码 | FORM-COMPREHENSIVE |
| fieldName | 英文字段名 | objectCode |
| label | 中文名称 | 对象编码 |
| type | 字段类型 | text / number / enum / date / coordinate / geometry / photo / attachment / relation |
| required | 是否必填 | 是 / 否 / 条件必填 |
| component | 前端控件 | 输入框、下拉框、地图点选、画线、画面、附件上传 |
| storageTarget | 入库目标 | health.health_object.objectCode |
| qcRules | 质检规则 | QC-P1-REQ-002 |
| description | 说明 | 与对象库编码一致 |
