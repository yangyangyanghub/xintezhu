# FreeCAD 13x11 农村别墅外观设计说明

## 已批准范围

- 执行路径：方案 A（参考图近似重建）
- 交付物：FCStd
- 尺寸假设：按默认值继续

## 目标

基于用户提供的单张农村自建房参考图，使用 FreeCAD 生成一个可编辑的三层别墅外观模型，重点还原：

- 三层整体体量
- 正立面主入口与门厅关系
- 二层与三层外挑阳台关系
- 四坡浅屋顶体量
- 主立面开窗节奏

本次不覆盖室内布局、结构节点、施工图深度和材料精算。

## 默认建模假设

- 建筑占地：13m × 11m
- 层数：3 层
- 一层层高：3.6m
- 二、三层层高：3.3m
- 女儿墙 / 檐口高度：0.9m
- 屋顶形式：四坡浅屋顶
- 单位：m

## 方法

1. 将参考图作为外观风格依据，而不是几何真值来源。
2. 先建立简化体块，再补主入口、阳台、立柱、窗洞和屋顶。
3. 以可编辑、可继续深化为优先，不追求一次性做满全部装饰线条。

## 约束与风险

- 单张透视图无法可靠反推出真实进深、窗洞厚度、檐口细部和结构做法。
- 当前 CLI 已确认支持建模、导出与保存，但不直接导入 PNG/JPG 作为参考平面。
- 因此本次模型属于“参考图近似重建”，不是施工级模型。

## 输出约定

- 目标文件：`exports/freecad-villa-13x11-exterior.fcstd`
- 若验证需要产生辅助文件，放在 `temp/` 或 `exports/`，不放根目录。
